# <a name="licensing-microsoft-365-for-small-and-medium-sized-businesses"></a>Lizensierung von Microsoft 365 für kleine und mittlere Unternehmen

Microsoft 365 Business Premium ist das best-in-class-Angebot von Microsoft für kleine und mittelständische Unternehmen (SMB). Microsoft 365 Business Premium ist eine integrierte Lösung, die  die Produktivität von Microsoft 365 **erweiterter Sicherheit** und **Geräteverwaltung vereint.**

In diesem Artikel werden Lizenzierungsoptionen für KMUs mit einem besonderen Schwerpunkt auf Microsoft 365 Business Premium.
