### <a name="distributed-key-manager"></a><span data-ttu-id="93150-157">Distributed Key Manager</span><span class="sxs-lookup"><span data-stu-id="93150-157">Distributed Key Manager</span></span>

<span data-ttu-id="93150-p115">Neben Bitlocker verwenden wir eine Technologie, die Distributed Key Manager (DKM) genannt wird. In Exchange Online, only certain service accounts under which the Exchange processes run are part of that security group.
