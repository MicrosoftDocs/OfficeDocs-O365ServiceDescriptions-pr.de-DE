# <a name="sharepoint-service-description"></a>SharePoint-Dienstbeschreibung

SharePoint in Microsoft 365 hilft Organisationen dabei, Inhalte, Wissen und Anwendungen zu teilen und zu verwalten, um:

- Fördern von Teamwork
- Schnelle Suche nach Informationen
- Nahtlose Zusammenarbeit innerhalb der Organisation

Eine kurze Übersicht finden Sie unter [Was ist SharePoint?](https://support.office.com/article/97b915e6-651b-43b2-827d-fb25777f446f)
  
Ausführliche Informationen zu Abonnements, die Benutzern SharePoint ermöglichen, finden Sie in der [vollständigen Vergleichstabelle](https://go.microsoft.com/fwlink/?linkid=2139145)für Abonnements. Informationen zu Microsoft 365 Systemanforderungen finden Sie unter [Microsoft 365 und Office Ressourcen.](https://www.microsoft.com/microsoft-365/microsoft-365-and-office-resources/)

Informationen zu SharePoint Features für CLOUD-Kunden von US Government finden Sie unter [SharePoint für US Government-Umgebungen.](../office-365-platform-service-description/office-365-us-government/sharepoint.md)
  
## <a name="whats-new"></a>Neuerungen

Informationen zu den neuesten Features, die veröffentlicht werden, finden Sie unter [Neuigkeiten in SharePoint](https://support.office.com/article/02449ef0-027e-4089-8717-f0ae7ea58029).
  
## <a name="subscription-options"></a>Abonnementoptionen

Microsoft 365 steht in einer Vielzahl von Plänen zur Verfügung, um die Anforderungen Ihrer Organisation optimal zu erfüllen. Ausführliche Informationen zu Abonnements, die Benutzern SharePoint ermöglichen, finden Sie in der [vollständigen Vergleichstabelle](https://go.microsoft.com/fwlink/?linkid=2139145)für Abonnements.
  
Um auf SharePoint zugreifen zu können, müssen Benutzern in Ihrer Organisation eine Lizenz zugewiesen werden, die SharePoint enthält. Ein Firstline-Plan wird nicht als eigenständiges Angebot verkauft, nur als Teil [Office 365 F3](https://www.microsoft.com/microsoft-365/business/office-365-f1), [Microsoft 365 F1](https://www.microsoft.com/microsoft-365/enterprise/f1?activetab=pivot:overviewtab) oder [Microsoft 365 F3.](https://www.microsoft.com/microsoft-365/enterprise/f3?activetab=pivot:overviewtab) 
  
## <a name="rights-of-guests"></a>Rechte von Gästen

Wenn Sie einen Plan erwerben und eine Website erstellen, die Unternehmensfeatures verwendet, erhalten Gäste, die Sie einladen, rechte zum Verwenden und/oder Anzeigen der Unternehmensfeatures innerhalb der Website, zu der Sie sie eingeladen haben. Sie können Gäste zwar einladen, eine vollständige Palette von Aktionen auf einer Website auszuführen, sie verfügen jedoch nicht über die gleichen Funktionen wie ein lizenzierter Benutzer in Ihrer Organisation. Wenn Ihr Plan beispielsweise Desktopversionen von Office-Anwendungen umfasst, können Gäste sie nur auf ihren eigenen Computern installieren, wenn Sie ihnen eine Lizenz zuweisen.

## <a name="onedrive"></a>OneDrive

OneDrive in Microsoft 365 ist der Onlinespeicher für einzelne Benutzer innerhalb einer Organisation. An diesem Ort können Personen ihre Arbeitsdaten bequem und sicher geräteübergreifend speichern, synchronisieren und freigeben. OneDrive ist in SharePoint enthalten und kann auch als eigenständiger Plan erworben werden. Informationen dazu, wie viel OneDrive Speicher in jedem Plan enthalten ist, finden Sie in der [OneDrive Dienstbeschreibung.](../onedrive-for-business-service-description.md)
