#### <a name="basic-client-limitations"></a>Einschränkungen für Basisclients

Die folgenden Features stehen in der vollständigen Clientversion zur Verfügung, jedoch nicht in der Basisversion des Clients:

- Einstellungen für Teamanruf verwalten
- Verwalten von Stellvertretungen    
- Anrufe im Namen eines anderen Kontakts tätigen (Vorgesetzter/Stellvertreter-Szenario)    
- Anrufe eines anderen Benutzers verwalten, wenn eine Delegierung konfiguriert wurde    
- Hohes Anrufaufkommen verwalten    
- Anruf an eine Reaktionsgruppe initiieren
- Anrufe parken
- Gruppenanrufannahme
    
### <a name="changing-or-mixing-plans"></a>Ändern oder Mischen von Plänen

Wenn sich die Anforderungen Ihrer Organisation ändern, müssen Sie möglicherweise Ihren Microsoft 365-Plan ändern. Sie können von Ihrem aktuellen Abonnement in ein anderes Abonnement wechseln: 
  
- **In der gleichen Dienst Familie:** Sie können beispielsweise von Microsoft 365 Business Basic zu Microsoft 365 Business Standard wechseln oder von Office 365 E1 auf Office 365 E3.
- **Aus einem eigenständigen Plan:** Sie können beispielsweise von Exchange Online Plan 1 in Office 365 E1 umsteigen.     
- **An eine andere Dienst Familie:** Sie können beispielsweise von Microsoft 365 Business Basic zu Office 365 E3 wechseln.     
Informationen zum Ändern von Abonnements und zum Wechseln von einer Testversion zu einem kostenpflichtigen Abonnement finden Sie unter [Wechseln zu einem anderen Microsoft 365-Plan oder-Abonnement](https://support.office.com/article/HA104031833).
  
Sie können Enterprise-, Business-und eigenständige Pläne (beispielsweise Exchange Online Plan 1) in einem einzelnen Konto kombinieren. Vorhandene Einschränkungen für die Anzahl der Sitze pro Plan ändern sich jedoch nicht. Beispielsweise können Sie bis zu 300 Sitze pro Plan sowohl für Microsoft 365 Business Basic und Microsoft 365 Business Standard als auch für eine unbegrenzte Anzahl von Benutzern in Exchange Online Plan 1 haben.
