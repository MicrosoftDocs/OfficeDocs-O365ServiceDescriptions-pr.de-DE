# <a name="visio-for-the-web-service-description"></a>Visio für das Web – Dienstbeschreibung

Mit Visio für das Web können Sie Visio-Diagramme im Web anzeigen, bearbeiten und freigeben. Beginnen Sie mit einem einfachen Diagramm, um allgemeine Ideen zum Ausdruck zu bringen, z. B. das Darstellen eines Plans, das Erstellen eines Vorschlags oder das Beschreiben einer Abfolge von Schritten in einem Prozess. Es ist einfach und einfach, erstklassige Diagramme mit häufig verwendeten Diagrammtypen und reichhaltigen Formsätzen zu erstellen. Visio für das Web können Benutzer anzeigen und zusammenarbeiten, indem Sie Diagramme freigeben und Kommentare in Dateien einfügen, die in SharePoint Online oder OneDrive for Business Online hochgeladen wurden.
  
Weitere Informationen und erste Schritte finden Sie unter [Visio Plan 1](https://products.office.com/visio/visio-online).
  
Hilfe bei der Verwendung von Visio für das Web finden Sie unter [Visio help & learning](https://support.office.com/visio).
