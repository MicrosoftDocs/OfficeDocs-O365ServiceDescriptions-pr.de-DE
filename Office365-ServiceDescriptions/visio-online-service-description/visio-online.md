# <a name="visio-for-the-web"></a>Visio für das Web

Visio für das Web erweitert Ihre Microsoft Visio auf den Browser, in dem Sie in der Cloud gespeicherte Diagramme erstellen und bearbeiten können. Microsoft 365 Kunden Können Diagramme anzeigen, drucken und freigeben und Kommentare von unterwegs einfügen. Wenn Sie über ein Visio verfügen, können Sie zusätzliche Features verwenden, z. B. die in den folgenden Abschnitten beschriebenen.
  
## <a name="apply-rich-formatting-to-text-and-shapes"></a>Anwenden umfangreicher Formatierung auf Text und Shapes

Bearbeitungsfeatures, z. B. mehr Farben, Farbverläufe, Effekte und Formatvorlagen, sind in Visio für das Web sowie in der Desktopanwendung verfügbar. Sie können auch ein Textfeld hinzufügen oder aus einem Katalog von Formen auswählen und Formatvorlagen anwenden, die Füll-, Gliederungs- und Schatteneffekte definieren.
  
## <a name="arrange-objects"></a>Anordnen von Objekten

Sie können Shapes und Textfelder verschieben, ihre Größe ändern, drehen, kippen oder in reihenfolgen, und Sie können die Gruppierung von Shapes aufheben, um sie einzeln in Visio web zu bearbeiten.
  
## <a name="build-mashup-solutions"></a>Erstellen von Mashup-Lösungen

Die Visio für die JavaScript Mashup-API im Web ermöglicht Entwicklern den Zugriff auf ein Visio Webdiagramm, dessen Seiten und seine Formen. Weitere Informationen finden Sie [unter Visio im Web: häufig gestellte Fragen](https://support.office.com/article/e6647040-2fca-42ec-9fa5-d16a4e39e0ee).
  
## <a name="create-diagrams"></a>Erstellen von Diagrammen

Sie können auf einfache Weise Diagramme in Visio web erstellen und in SharePoint oder OneDrive for Business. Sie können sie dann für andere freigeben.
  
## <a name="cut-copy-and-paste"></a>Ausschneiden, Kopieren und Einfügen

Ausschneiden, Kopieren und Einfügen von Features sind in Visio web verfügbar. Sie können diese Features verwenden, um Formen und Text in einem Diagramm zu verschieben oder zu entfernen.
  
## <a name="edit-diagrams"></a>Bearbeiten von Diagrammen

Visio für das Web umfasst umfangreiche Bearbeitung, die Sie verwenden können, um Änderungen am Diagramm vorzunehmen. Sie können auch das Design Des Diagramms mit einer Vielzahl von Designfarben und Layouts ändern.
  
## <a name="embed-diagram-in-a-sharepoint-web-or-blog-page"></a>Einbetten von Diagrammen in eine SharePoint-, Web- oder Blogseite

Sie können Visio web verwenden, um ein Visio-Diagramm (das in der Visio-Desktopanwendung erstellt wurde) auf einer SharePoint-, Web- oder Blogseite einfügen. Weitere Informationen finden Sie [unter Visio im Web: häufig gestellte Fragen](https://support.office.com/article/e6647040-2fca-42ec-9fa5-d16a4e39e0ee).
  
## <a name="find"></a>Suchen

In Visio für das Web können Sie nach jedem Vorkommen eines Worts, einer Zahl oder eines Ausdrucks auf der Seite suchen, indem Sie Mehr **(die** Ellipsenschaltfläche) auswählen und dann **Suchen** auswählen (oder STRG+F drücken). Die Ergebnisse werden neben Ihrem Diagramm angezeigt. Visio durchsucht Shapenamen, Shapetext, Shapedaten und benutzerdefinierte Zellen.
  
## <a name="font-formatting"></a>Schriftartformatierung

Sie können die Attribute Schriftart, Schriftgrad und andere Schriftarten (z. B. Fett, Kursiv, Hochgestellt oder Unterschrift) in Visio web ändern.
  
## <a name="hyperlinks"></a>Hyperlinks

In Visio web können Sie direkt auf Hyperlinks zugreifen, indem Sie das Shape auswählen.
  
## <a name="navigation"></a>Navigation

In Visio für das Web können Sie zu anderen Seiten wechseln, indem Sie die Registerkarten unten links auswählen.
  
## <a name="office-add-ins"></a>Office-Add-Ins

Office add-ins sind noch nicht in Visio für das Web verfügbar.
  
## <a name="offline-viewing"></a>Offlineansichten

Nicht verfügbar in Visio für das Web. Office für das Web erfordert eine Internetverbindung und einen Internetbrowser. Sie benötigen die Visio Desktopanwendung, die auf Ihrem Computer installiert ist, um ein Diagramm anzeigen und bearbeiten zu können, während die Verbindung mit dem Internet oder dem lokalen Web Apps Server ihrer Organisation Office ist. Wenn Sie die Visio installiert haben, können Sie die gesamte Funktionalität von Visio verwenden, um Ihr Diagramm offline zu sehen und zu bearbeiten. Beim Speichern wird das Onlinediagramm mit Ihren Änderungen aktualisiert. Sie können auch eine Offlinekopie des Diagramms herunterladen und speichern; diese wird aber nicht mit der Onlineversion synchronisiert sein. Weitere Informationen zu Office Web Apps Server finden Sie [unter Office Web Apps Server](/webappsserver/how-office-web-apps-work-on-premises-with-sharepoint-2013).
  
## <a name="print-to-pdf"></a>Als PDF drucken

Sie können als PDF- oder Drucker mit Visio für das Web drucken.
  
## <a name="proofing-tools"></a>Korrekturhilfen

Korrekturhilfen wie Rechtschreibprüfung, Übersetzung und benutzerdefinierte Wörterbücher sind in der Visio für das Web nicht verfügbar. Es sind jedoch eine Barrierefreiheitsprüfung und Alttextbeschreibungen verfügbar.
  
## <a name="real-time-co-authoring"></a>Gemeinsame Dokumenterstellung in Echtzeit

Noch nicht verfügbar in Visio für das Web. Mit Visio für das Web können Sie Kommentare anzeigen, hinzufügen oder löschen.
  
## <a name="save-as-or-download-a-copy"></a>"Speichern unter" oder "Kopie herunterladen"

Visio für das Web können Sie Speichern unter verwenden oder eine Kopie Des Diagramms herunterladen, aber Sie müssen über die Visio-Desktopanwendung verfügen, um eine lokale Kopie zu bearbeiten.
  
## <a name="share-a-diagram"></a>Freigeben eines Diagramms

Wenn Ihr Diagramm in einer SharePoint-Dokumentbibliothek oder in OneDrive for Business gespeichert ist, ist Ihr Diagramm online, und Sie können es freigeben, indem Sie den betreffenden Personen statt einer E-Mail-Anlage einen Link zu dem Diagramm senden. Personen mit den erforderlichen Berechtigungen können das Diagramm in ihrem Webbrowser oder auf ihrem mobilen Gerät anzeigen. Wählen **Sie** oben rechts einfach Freigeben aus. Es wird empfohlen, einen Link zu teilen, um das Erstellen doppelt mit Daten verbundenen Diagrammen zu vermeiden.
  
## <a name="smartart"></a>SmartArt

Mit Visio für das Web können Sie einem Diagramm keine SmartArt hinzufügen. Wenn SmartArt jedoch mithilfe der Visio hinzugefügt wurde, wird es in der Visio für das Web angezeigt.
  
## <a name="tell-me"></a>Sie wünschen...

Das Feature "Mir sagen" ist in Visio für das Web nicht verfügbar. In der Visio-Desktopanwendung können Sie die Funktion „Sie wünschen..." verwenden, um schnell Hilfe zu einer Aufgabe zu finden, die Sie ausführen möchten.
  
## <a name="undo-and-redo"></a>Rückgängigmachen und Wiederholen

Die Funktionen Undo und Redo sind in Visio web verfügbar.
  
## <a name="view-and-add-comments"></a>Anzeigen und Hinzufügen von Kommentaren

 In Visio für das Web können Sie mit anderen zusammenarbeiten, indem Sie Kommentare in Ihrem Diagramm anzeigen, hinzufügen und darauf antworten. 
  
## <a name="view-shape-data"></a>Anzeigen von Shapedaten

Sie können die Visio für das Web verwenden, um die Metadaten für ein Diagramm zu sehen: Wählen Sie einfach die Form aus, und wählen Sie dann **Shape Info aus.**
  
## <a name="view-visio-files-in-the-browser"></a>Anzeigen von Visio-Dateien im Browser

Da Ihr Visio-Diagramm online gespeichert ist, können Sie einen Link zu Ihrem Diagramm senden. Personen können es in einem beliebigen Browser anzeigen. Es ist nicht erforderlich, dass die Visio oder eine Visio installiert ist, um ein Diagramm im Browser anzuzeigen.
  
## <a name="zoom"></a>Zoom

Visio für das Web enthält einen Schieberegler, mit dem Sie ein- und auszoomen können. Darüber hinaus können Sie das Feature &amp; Schwenkzoom verwenden, um einen Bereich des Diagramms zu vergrößern.
