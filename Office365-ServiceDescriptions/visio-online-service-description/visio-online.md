# <a name="visio-online"></a>Visio Online

Mit Visio Online können Sie Ihre Arbeit in Microsoft Visio auf den Webbrowser ausweiten, wo Sie mit Diagrammen direkt auf der Website arbeiten können, auf der sie gespeichert sind. Microsoft Office 365-Kunden mit Office Online können unterwegs Diagramme anzeigen und freigeben sowie Kommentare einfügen.
  
## <a name="apply-rich-formatting-to-text-and-shapes"></a>Anwenden umfangreicher Formatierung auf Text und Shapes
<a name="BM_1"> </a>

Nicht verfügbar in Visio Online. In der Visio-Desktopanwendung sind erweiterte Formatierungsfeatures verfügbar, z. B. weitere Farben, Farbverläufe, Effekte und Formatvorlagen. Mit Visio Online können Sie ein Textfeld hinzufügen oder eine Auswahl aus einem Katalog mit Shapes treffen und Formatvorlagen anwenden, in denen Füllung, Gliederung und Schatteneffekte vorgegeben sind.
  
## <a name="arrange-objects"></a>Anordnen von Objekten
<a name="BM_2"> </a>

Derzeit nicht verfügbar in Visio Online. In der Visio-Desktopanwendung können Sie Shapes und Textfelder verschieben, drehen, neu anordnen oder ihre Größe ändern sowie die Gruppierung von Shapes aufheben, um sie einzeln zu bearbeiten. 
  
## <a name="build-mashup-solutions"></a>Erstellen von Mashup-Lösungen
<a name="BM_3"> </a>

Mit der Visio Online JavaScript Mashup-API können Entwickler auf Visio-Webdiagramme und die zugehörigen Seiten und Shapes zugreifen und diese bearbeiten. Weitere Informationen finden Sie unter [Häufig gestellte Fragen zu Visio Online](https://go.microsoft.com/fwlink/?linkid=825706).
  
## <a name="create-diagrams"></a>Erstellen von Diagrammen
<a name="BM_4"> </a>

In Visio Online können Sie keine Diagramme erstellen. Sie können allerdings Diagramme in der Visio-Desktopanwendung erstellen und sie anschließend in SharePoint oder OneDrive for Business hochladen. Anschließend können Sie diese Diagramme für andere Benutzer freigeben.
  
## <a name="cut-copy-and-paste"></a>Ausschneiden, Kopieren und Einfügen
<a name="BM_5"> </a>

Derzeit nicht verfügbar in Visio Online. Sie können die Visio-Desktopanwendung zum Ausschneiden, Kopieren und Einfügen von Funktionen zum Verschieben oder Entfernen von Shapes und Text in einem Diagramm verwenden.
  
## <a name="edit-diagrams"></a>Bearbeiten von Diagrammen
<a name="BM_6"> </a>

In Visio Online können Sie keine Änderungen an einem Diagramm vornehmen. Wenn Sie Änderungen an einem Diagramm vornehmen müssen, verwenden Sie die Visio-Desktopanwendung.
  
## <a name="embed-diagram-in-a-sharepoint-web-or-blog-page"></a>Einbetten von Diagrammen in eine SharePoint-, Web- oder Blogseite
<a name="BM_7"> </a>

Mit Visio Online können Sie ein Visio-Diagramm (in der Visio-Desktopanwendung erstellt) auf einer SharePoint-, Web- oder Blogseite einfügen. Weitere Informationen finden Sie unter [Häufig gestellte Fragen zu Visio Online](https://go.microsoft.com/fwlink/?linkid=825706).
  
## <a name="find"></a>Suchen
<a name="BM_8"> </a>

In Visio Online können Sie nach jedem Vorkommen eines Worts, einer Zahl oder eines Ausdrucks auf der Seite suchen, indem Sie auf die Schaltfläche „..." klicken und „Suchen" auswählen (oder STRG+F drücken). Die Ergebnisse werden neben Ihrem Diagramm angezeigt. Visio durchsucht Shapenamen, Shapetext, Shapedaten und benutzerdefinierte Zellen.
  
## <a name="font-formatting"></a>Schriftartformatierung
<a name="BM_9"> </a>

Sie können die Schriftart, Schriftgröße oder andere Schriftartenattribute (z. B. Fett, Kursiv, Hochgestellt oder Unterstrichen) in Visio Online nicht ändern.
  
## <a name="hyperlinks"></a>Hyperlinks
<a name="BM_10"> </a>

In Visio Online können Sie direkt auf die Links zugreifen, indem Sie auf das Shape klicken.
  
## <a name="navigation"></a>Navigation
<a name="BM_11"> </a>

In Visio Online können Sie zu anderen Seiten navigieren, indem Sie auf die Registerkarten in der unteren linken Ecke klicken.
  
## <a name="office-add-ins"></a>Office-Add-Ins
<a name="BM_12"> </a>

Office-Add-Ins sind in Visio Online nicht verfügbar.
  
## <a name="offline-viewing"></a>Offlineansichten
<a name="BM_13"> </a>

Nicht verfügbar in Visio Online. Für Office Online sind eine Internetverbindung und ein Webbrowser erforderlich. Sie müssen auf Ihrem Computer die Visio-Desktopanwendung installiert haben, um ein Diagramm anzeigen und bearbeiten zu können, wenn keine Verbindung mit dem Internet oder dem lokalen Office Web Apps Server Ihrer Organisation besteht. Wenn Sie die Visio-Desktopanwendung installiert haben, können Sie alle Visio-Funktionen zum Anzeigen und Bearbeiten Ihres Diagramms verwenden. Beim Speichern wird das Onlinediagramm mit Ihren Änderungen aktualisiert. Sie können auch eine Offlinekopie des Diagramms herunterladen und speichern; diese wird aber nicht mit der Onlineversion synchronisiert sein. Erfahren Sie mehr über [Office Web Apps Server](https://technet.microsoft.com/library/ff431685.aspx).
  
## <a name="print-to-pdf"></a>Als PDF drucken
<a name="BM_14"> </a>

Die Funktion „Als PDF drucken" ist in Visio Online nicht verfügbar.
  
## <a name="proofing-tools"></a>Korrekturhilfen
<a name="BM_15"> </a>

Korrekturhilfen wie z. B. Rechtschreibprüfung, Übersetzung und benutzerdefinierte Wörterbüchern sind in Visio Online nicht verfügbar.
  
## <a name="real-time-co-authoring"></a>Gemeinsame Dokumenterstellung in Echtzeit
<a name="BM_16"> </a>

In Visio Online nicht verfügbar. Erweiterte Bearbeiterfeatures, z. B. zum Bearbeiten von Konflikten bei der Zusammenführung oder zum Vergleichen von Diagrammen, sind nur in der Visio-Desktopanwendung verfügbar. Mit Visio Web App können Sie Kommentare anzeigen, hinzufügen oder löschen.
  
## <a name="save-as-or-download-a-copy"></a>"Speichern unter" oder "Kopie herunterladen"
<a name="BM_17"> </a>

Mit Visio Online können Sie die Funktion „Speichern unter" verwenden oder eine Kopie des Diagramms herunterladen, zum Bearbeiten einer lokalen Kopie benötigen Sie jedoch die Microsoft Visio-Desktopanwendung.
  
## <a name="share-a-diagram"></a>Freigeben eines Diagramms
<a name="BM_18"> </a>

Wenn Ihr Diagramm in einer SharePoint-Dokumentbibliothek oder in OneDrive for Business gespeichert ist, ist Ihr Diagramm online, und Sie können es freigeben, indem Sie den betreffenden Personen statt einer E-Mail-Anlage einen Link zu dem Diagramm senden. Personen mit den erforderlichen Berechtigungen können das Diagramm in ihrem Webbrowser oder auf ihrem mobilen Gerät anzeigen. Klicken Sie in der rechten oberen Ecke auf **Freigeben**. 
  
## <a name="smartart"></a>SmartArt
<a name="BM_19"> </a>

In Visio Online können Sie keine SmartArt-Grafiken zu einem Diagramm hinzufügen. Wenn SmartArt-Grafiken jedoch mit der Visio-Desktopanwendung hinzugefügt wurden, werden sie in Visio Online angezeigt.
  
## <a name="tell-me"></a>Funktion "Was möchten Sie tun?"
<a name="BM_20"> </a>

Die Funktion „Sie wünschen..." ist in Visio Online nicht verfügbar. In der Visio-Desktopanwendung können Sie die Funktion „Sie wünschen..." verwenden, um schnell Hilfe zu einer Aufgabe zu finden, die Sie ausführen möchten.
  
## <a name="undo-and-redo"></a>"Rückgängig" und "Wiederholen"
<a name="BM_21"> </a>

Die Funktionen „Rückgängig" und „Wiederholen" sind in Visio Online nicht verfügbar.
  
## <a name="view-and-add-comments"></a>Anzeigen und Hinzufügen von Kommentaren
<a name="BM_22"> </a>

 In Visio Online können Sie mit anderen Personen zusammenarbeiten, indem Sie Kommentare in Ihrem Diagramm anzeigen, hinzufügen oder auf diese antworten. 
  
## <a name="view-shape-data"></a>Anzeigen von Shapedaten
<a name="BM_23"> </a>

In Visio Online können Sie die Metadaten für ein Diagramm anzeigen. Klicken Sie dazu auf das Shape, und klicken Sie dann auf **Shape-Info**.
  
## <a name="view-visio-files-in-the-browser"></a>Anzeigen von Visio-Dateien im Browser
<a name="BM_24"> </a>

Da Ihr Visio-Diagramm online gespeichert ist, können Sie einen Link zu Ihrem Diagramm senden. Personen können es in einem beliebigen Browser anzeigen. Sie müssen nicht über die Visio-Desktopanwendung verfügen.
  
## <a name="zoom"></a>Zoom
<a name="BM_25"> </a>

Visio Online includes a slider that lets you zoom in and out. Also, you can use the Pan &amp; Zoom feature to zoom in on a region of the diagram.
