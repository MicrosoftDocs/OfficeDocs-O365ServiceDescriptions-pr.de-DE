# <a name="word-for-the-web"></a>Word für das Internet

Word für das Web (ehemals Word Web App) erweitert Ihre Microsoft Word Erfahrung auf den Webbrowser, wo Sie mit Dokumenten direkt auf der Website arbeiten können, auf der das Dokument gespeichert ist. 
  
## <a name="alignment-guides-and-live-layout"></a>Hilfslinien und Echtzeitlayout
<a name="bkmk_AlignmentGuidesLivelayout"> </a>

Lineale und Gitternetzlinien stehen in Word für das Internet nicht zur Verfügung. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="apply-styles"></a>Anwenden von Formatvorlagen
<a name="bkmk_Styles"> </a>

Die in einem Dokument gespeicherten Absatz-und Zeichenformate stehen zur Verfügung, um auf Text in Word für das Internet angewendet zu werden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="office-add-ins"></a>Office-Add-Ins
<a name="bkmk_AppsforOffice"> </a>

Sie können im Office Store Add-Ins für Word erwerben. Office-Add-Ins funktionieren nur in der Word-Desktop-App und stehen in Word für das Internet nicht zur Verfügung. Informationen zum Abrufen eines Add-Ins für Word finden Sie [hier](https://go.microsoft.com/fwlink/?LinkId=391443).
  
## <a name="autosave"></a>AutoSpeichern
<a name="bkmk_Save"> </a>

Word für das Internet speichert Ihr Dokument automatisch, wenn Sie Änderungen vornehmen.
  
## <a name="autotext"></a>AutoText
<a name="bkmk_autotext"> </a>

Mithilfe von AutoText können Sie aus dem AutoText-Katalog Bausteine aus vorformatiertem Text einfügen. Dies ist beispielsweise sinnvoll, wenn Sie denselben Textblock wiederholt eingeben müssen und der Text viel Formatierung enthält. AutoText-Einträge werden durch Hinzufügen von ausgewähltem Text zum AutoText-Katalog erstellt. Weitere Informationen hierzu finden Sie unter [Automatisches Einfügen von Text](https://go.microsoft.com/fwlink/?linkid=846739).
  
## <a name="browse-and-start-from-professionally-designed-templates"></a>Suchen nach und Verwenden von professionell gestalteten Vorlagen
<a name="bkmk_DesignedTemplates"> </a>

Nicht verfügbar in Word für das Internet. Erweiterte Designfeatures, z. B. zum Erstellen von Dokumenten auf Grundlage einer der zahlreichen professionell gestalteten Vorlagen von Office.com, sind nur in der Word-Desktop-App verfügbar. Office 365 Kunden können jedoch eine Dokumentbibliothek so konfigurieren, dass eine benutzerdefinierte Vorlage gestartet wird, wenn ein Benutzer ein neues Dokument erstellt. Weitere Informationen finden Sie unter [Hinzufügen eines Inhaltstyps zu einer Liste oder Bibliothek](https://go.microsoft.com/fwlink/p/?LinkId=391446).
  
## <a name="bullets-and-numbering"></a>Aufzählung und Nummerierung
<a name="bkmk_BulletsNumbering"> </a>

Mit Word für das Internet können Sie eine Auswahl von drei Aufzählungszeichenformat Vorlagen oder fünf Nummerierung Formatvorlagen anwenden. Klicken Sie auf die Schaltflächen "Einzug vergrößern" und "Einzug verkleinern", um die Listenebene für vorhandene Aufzählungen und nummerierte Listen im Dokument sowie für in der Word-Desktop-App erstellte Aufzählungen und Nummerierungen zu ändern. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="captions"></a>Beschriftungen
<a name="bkmk_Captions"> </a>

Nicht verfügbar in Word für das Internet. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie einem Dokument in Word Beschriftungen hinzuzufügen. Erfahren Sie mehr über das [Hinzufügen von Beschriftungen in Word 2013](https://go.microsoft.com/fwlink/p/?LinkId=282312).
  
## <a name="citations-and-bibliography"></a>Zitate und Literaturverzeichnis
<a name="bkmk_Citations"> </a>

Nicht verfügbar in Word für das Internet. Sie können vorhandene Zitate in Word für das Internet anzeigen, aber keine Zitate hinzufügen. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie Ihrem Dokument in Word ein Literaturverzeichnis und Zitate hinzufügen. Erfahren Sie mehr über das [Erstellen eines Literaturverzeichnisses in Word 2013](https://go.microsoft.com/fwlink/p/?LinkId=282314). 
  
## <a name="clipboard"></a>Zwischenablage
<a name="bkmk_Clipboard"> </a>

Aufgrund der Einschränkungen der Webbrowser Technologie unterscheidet sich das Kopieren und Einfügen von Text in Office für das Internet vom Kopieren und Einfügen von Text in Office-Desktopanwendungen. Erfahren Sie mehr über [Kopieren und Einfügen in Office für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=282321).
  
## <a name="collaboration--advanced"></a>Erweiterte Zusammenarbeit
<a name="bkmk_AdvancedCollaboration"> </a>

Nicht verfügbar in Word für das Internet. Erweiterte Features für die Zusammenarbeit, z. B. zum Aktivieren der Nachverfolgung von Änderungen und das Zusammenführen, Vergleichen und Zusammenfassen von Dokumenten, sind nur in der Word-Desktop-App verfügbar. Wie erwartet werden nachverfolgte Änderungen in Word für das Internet angezeigt, während Sie sich im Ansichtsmodus befinden.
  
## <a name="content-controls"></a>Inhaltssteuerelemente
<a name="bkmk_contentcontrols"> </a>

Inhaltssteuerelemente sind einzelne Steuerelemente, die Sie für die Verwendung in Vorlagen, Formularen und Dokumenten hinzufügen und anpassen können. Weitere Informationen zu Inhaltssteuerelementen finden Sie unter [Inhaltssteuerelemente](https://go.microsoft.com/fwlink/p/?LinkId=846741).
  
## <a name="cover-pages"></a>Deckblätter
<a name="bkmk_coverpage"> </a>

Microsoft Word bietet einen Katalog von vorgefertigten Deckblättern. Sie können einfach ein Deckblatt auswählen und den Beispieltext durch Ihren eigenen Text ersetzen.
  
## <a name="creation-of-advanced-document-elements"></a>Erstellen erweiterter Dokumentelemente
<a name="bkmk_Creationadvanceddocumentelements"> </a>

Nicht verfügbar in Word für das Internet. Erweiterte Dokumentelemente wie AutoText, Deckblätter, Literaturverzeichnis, Inhaltsverzeichnis, Index, Formeln sowie Wasserzeichen sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="document-parts-and-themes"></a>Dokumentbausteine und Designs
<a name="bkmk_Documentpartsandthemes"> </a>

Nicht verfügbar in Word für das Internet. Dokumentbausteine und Designs wie Inhaltssteuerelemente und vorformatierte Sammlungen aus Farbschema, Schriftartenschema und Voreinstellungen für grafische Effekte sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="document-translation"></a>Übersetzung von Dokumenten
<a name="bkmk_DocumentTranslation"> </a>

Übersetzung von Dokumenten direkt im Browser. Sogar Dokumentkommentare werden übersetzt.
  
## <a name="dropbox"></a>Dropbox
<a name="bkmk_Dropbox"> </a>

Bei Dropbox handelt es sich um einen Dateihostingdienst, der einen Cloudspeicher, eine Dateisynchronisierung, eine persönliche Cloud und Clientsoftware bietet.
  
## <a name="equations"></a>Formeln
<a name="bkmk_equations"> </a>

Microsoft Word enthält integrierte Unterstützung zum Schreiben und Bearbeiten von Formeln sowie Gleichungen, die Sie in Ihren Dokumenten ablegen können.
  
## <a name="find-and-replace"></a>Suchen und Ersetzen
<a name="bkmk_Find"> </a>

Suchen und Ersetzen von Text in Ihrem Dokument mithilfe des Tools suchen und ersetzen in Word für die Webanwendung und Word-Desktop-App. Weitere Informationen zum Verwenden von suchen und ersetzen in Word für das Internet finden Sie unter [What es New in Word for the Internet](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="font-formatting"></a>Schriftartformatierung
<a name="bkmk_Fontformatting"> </a>

Mit Word für das Internet können Sie Schriftart, Schriftgrad und verschiedene Schriftart Formatierungsattribute anwenden, einschließlich Fett, kursiv, unterstrichen und hochgestellt. Markieren Sie Text, ändern Sie die Schriftfarbe, und löschen Sie die Formatierung für markierten Text. Die Formatierung von Dokumentdesigns steht zum Formatieren von Text in Word für das Internet zur Verfügung. Suchen Sie die aktiven Designschriftarten oben im Katalog "Schriftarten" und "Designfarben" im Schrift Farbkatalog. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="footnotes-and-endnotes"></a>Fußnoten und Endnoten
<a name="bkmk_Footnotes"> </a>

Sie können Ihren Dokumenten in Word für das Internet Fußnoten und Endnoten hinzufügen. Erfahren Sie mehr über [das Hinzufügen von Fußnoten und Endnoten in Word für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=397306).
  
## <a name="full-fidelity-reading-view"></a>Originalgetreue Leseansicht
<a name="bkmk_FullFidelityReadingView"> </a>

Wenn Sie Ihr Dokument in OneDrive oder SharePoint öffnen, wird das Dokument im Browser von Word für das Internet geöffnet. Layout und Formatierung entsprechen dem, was Sie sehen, wenn Sie das Dokument in der Word-Desktop-App in der Drucklayoutansicht öffnen.
  
## <a name="graphic-effect-presets"></a>Voreinstellungen für grafische Effekte
<a name="bkmk_presets"> </a>

Sie können Voreinstellungen verwenden, um Spezialeffekte, wie z. B. Drehung, Schatten, Sättigung und Texteffekte, auf Grafiken anzuwenden.
  
## <a name="headers-and-footers"></a>Kopf- und Fußzeilen
<a name="bkmk_WordHeadersandFooters"> </a>

Fügen Sie eine Kopf-und Fußzeile in das Dokument sowohl in Word für das Internet als auch in der Word-Desktop-App ein. Sie haben auch die Möglichkeit, Kopf- und Fußzeilen auf alle Seiten außer die erste Seite eines Dokuments anzuwenden. Weitere Informationen zum Verwenden von Kopf-und Fußzeilen in Word für das Internet finden Sie unter [What es New in Word for the Internet](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="hyperlinks"></a>Hyperlinks
<a name="bkmk_Hyperlinks"> </a>

Mit Word für das Internet können Sie auf einfache Weise Links zu Textmarken in einem Dokument erstellen und eine Bildschirm Spitze festlegen, die angezeigt wird, wenn der Cursor auf den Link zeigt. Erfahren Sie mehr über [Hyperlinks in Word für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=271863).
  
## <a name="index"></a>Index 
<a name="bkmk_index"> </a>

Ein Index enthält die Begriffe und Themen, die in einem Dokument erläutert werden, zusammen mit den Seiten, auf denen diese angezeigt werden. Um einen Index zu erstellen, markieren Sie die Indexeinträge, indem Sie den Namen des Haupteintrags und den Querverweis in Ihrem Dokument bereitstellen und dann den Index erstellen.
  
## <a name="ink-editor"></a>Freihand-Editor
<a name="BKMK_InkEditor"> </a>

Mit dem frei Hand Editor können Sie Dokumente mit natürlichen Gesten wie etwa Kreisen bearbeiten und formatieren, um Text zu markieren und zu streichen.
  
## <a name="insert-online-pictures"></a>Einfügen von Onlinebildern
<a name="bkmk_InsertPicturessClipArt"> </a>

Mit Word für das Internet können Sie ein Bild aus einer auf Ihrer lokalen Festplatte gespeicherten Datei oder aus Bing-Bildern einfügen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="irm-and-password-protection"></a>IRM- und Kennwortschutz
<a name="bkmk_IRMpasswoprdprotect"> </a>

Word für das Internet zeigt Dokumente an, die mit der Verwaltung von Informationsrechten (Information Rights Management, IRM) geschützt sind. Diese Dokumente können jedoch nicht im Browser bearbeitet werden, und Sie können IRM-geschützte Dokumente nicht in Word für das Internet erstellen. Word für das Internet kann keine Dokumente öffnen, die mit einem Kennwort verschlüsselt sind. Erweiterte Funktionen für den Dokumentschutz, z. B. das Erstellen IRM-geschützter Dokumente und das Anwenden von Kennwortschutz, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="learning-tools"></a>Lerntools
<a name="BKMK_LearningTools"> </a>

Lern Tools sind so konzipiert, dass Sie Ihre Lesekompetenz verbessern können, indem Sie die Möglichkeit zum korrekten aussprechen von Wörtern, zum schnellen und genauen lesen und zur besseren Lesbarkeit der Wörter stärken. 
  
## <a name="lists"></a>Listen
<a name="BKMK_Lists_WordOnline"> </a>

Beginnen Sie in Word für das Internet mit der Eingabe direkt unter einer vorhandenen nummerierten Liste, und die nächste Zeile wird automatisch Teil der Liste. Erfahren Sie mehr über [das Erstellen einer Liste in Word für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=397307).
  
## <a name="mail-merge"></a>Seriendruck
<a name="bkmk_Mailmerge"> </a>

Nicht verfügbar in Word für das Internet. Erweiterte Features für die Datenintegration, z. B. Seriendruck unter Verwendung einer Excel-Tabelle, sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="offline-viewing-and-authoring"></a>Offlineansicht und -erstellung
<a name="bkmk_Offlineviewingauthoring"> </a>

Nicht verfügbar in Word für das Internet. Office für das Web erfordert eine Internetverbindung und einen Internet Browser. Sie müssen auf Ihrem Computer die Word-Desktop-App installiert haben, um ein Dokument anzeigen und bearbeiten zu können, während keine Verbindung mit dem Internet oder dem lokalen Office Web Apps Server Ihrer Organisation besteht. Wenn Sie die Word-Desktop-App installiert haben, können Sie alle Word-Funktionen zum Anzeigen und Bearbeiten Ihres Dokuments verwenden. Beim Speichern wird das Onlinedokument mit Ihren Änderungen aktualisiert. Sie können auch eine Offlinekopie des Dokuments herunterladen und speichern; diese wird aber nicht mit der Onlineversion synchronisiert sein. Erfahren Sie mehr über [Office Web Apps Server](https://go.microsoft.com/fwlink/p/?LinkId=282318).
  
## <a name="page-breaks"></a>Seitenwechsel
<a name="bkmk_PageBreaks"> </a>

Einfügen von Seitenumbrüchen in Word für das Internet und die Word-Desktop-App.

Mit Word für das Internet können Sie die Ränder, die Papiergröße anpassen und festlegen, ob die Seite vertikal oder horizontal in Ihren Dokumenten angeordnet ist. In Word für das Internet werden keine Lineale und Gitternetzlinien angezeigt. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859). 
  
Hier erfahren Sie, wie Sie [eine leere Seite in Word für das Internet löschen](https://support.office.com/en-US/article/Delete-a-blank-page-B583020A-D26C-4661-873C-CB66FBDB03D1).

<a name="bkmk_PageNumbers"> </a>

Einfügen von Seitenzahlen in das Dokument sowohl in Word für das Internet als auch in der Word-Desktop-App. Sie haben auch die Möglichkeit, Seitenzahlen auf alle Seiten außer die erste Seite Ihres Dokuments anzuwenden. Weitere Informationen zum Verwenden von Seitenzahlen in Word für das Internet finden Sie unter [What es New in Word for the Internet](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="paragraph-formatting"></a>Absatzformatierung
<a name="bkmk_ParagraphFormatting"> </a>

Mit Word für das Internet können Sie Absätze nach links, rechts oder zentriert ausrichten. Ausführen von Text von rechts nach links; Einzug vergrößern oder verkleinern; und Formatieren von Absätzen als Aufzählung oder nummerierte Liste. Sie können die Formatierung auch löschen. Sie können den Zeilenabstand in Word für das Internet nicht anpassen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="picture-tools"></a>Bildtools
<a name="bkmk_PictureTools"> </a>

Mit Word für das Internet können Sie die Größe der Bilder ändern und in Ihrem Dokument alternativen Text hinzufügen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="preformatted-font-and-color-schemes"></a>Vorformatierte Schriftarten-und Farbschemas
<a name="bkmk_themes"> </a>

Sie können Designs verwenden, um den Prozess des Erstellens ansprechender, professionell aussehender Dokumente zu vereinfachen. Designs können auf Tabellen, Diagramme, Formen und Diagramme angewendet werden, sodass einheitliche Schriften, Effekte und Farbschemas entstehen.
  
## <a name="present-online"></a>Onlinepräsentation
<a name="bkmk_Presentonline"> </a>

Nicht verfügbar in Word für das Internet. Mit der Word 2013-Desktop-App können Sie den Office-Präsentationsdienst oder Skype for Business Online verwenden, um ein Dokument während einer Onlinebesprechung anzuzeigen, sodass Ihre Kollegen einem Link folgen können, um zu sehen, wie Sie durch das Dokument blättern. 
  
## <a name="print-to-pdf"></a>Als PDF drucken
<a name="bkmk_PrintPDF"> </a>

Mit Word für das Internet erhalten Sie einen Prozess mit einem Mausklick zum Generieren einer PDF-Datei und zum Anzeigen des Dialogfelds Drucken. Erfahren Sie mehr über [das Drucken eines Dokuments in Word für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=271860).
  
## <a name="printing--advanced"></a>Erweiterte Druckfunktionen
<a name="bkmk_AdvancedPrinting"> </a>

Nicht verfügbar in Word für das Internet. Erweiterte Druckfeatures, z. B. zum Drucken von Markups und Dokumenteigenschaften, sind nur in der Word-Desktop-App verfügbar 
  
## <a name="proofing-tools"></a>Korrekturhilfen
<a name="bkmk_ProofingTools"> </a>

Word für das Internet überprüft automatisch die Rechtschreibung während der Eingabe und wendet eine wellenförmige rote Unterstreichung auf falsch geschriebenen Text an. Allgemeine AutoKorrektur-Aktionen sind enthalten, z. B. die Korrektur häufig vorkommender Rechtschreibfehler oder die Konvertierung von Zeichen in Symbole. Außerdem können Sie die Sprache für die Korrekturhilfe ändern oder die Rechtschreibprüfung für markierten Text deaktivieren. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="proofing-tools--advanced"></a>Erweitere Korrekturhilfen
<a name="bkmk_AdvancedProofingTools"> </a>

Nicht verfügbar in Word für das Internet. Erweitere Korrekturhilfen, z. B. zum Hinzufügen neuer Wörter zum Wörterbuch für die Rechtschreibprüfung, zum Anwenden eines benutzerdefinierten Wörterbuchs sowie Sprach- und Übersetzungstools, sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="real-time-co-authoring"></a>Gemeinsame Dokumenterstellung in Echtzeit
<a name="bkmk_Coauthoring"> </a>

Mehr als eine Person kann gleichzeitig in einem Dokument arbeiten. In Word für das Internet und in Word 2016 können Sie in Echtzeit sehen, wo Ihre Mitverfasser in dem Dokument arbeiten, damit Sie bei der Bearbeitung keine Konflikte verursachen, und die Änderungen werden angezeigt, wenn Sie gerade erstellt werden. Word 2013 unterstützt Gleichzeitiges Bearbeiten, aber es gibt keine Anwesenheits Angabe, und Änderungen können von mehreren Autoren erst angezeigt werden, wenn das Dokument gespeichert wird. Weitere Informationen zur gemeinsamen Erstellung in Echtzeit finden Sie unter [What es New in Word for the Internet](https://go.microsoft.com/fwlink/?LinkId=389520) und [Zusammenarbeit an Word-Dokumenten mit der gemeinsamen Erstellung in Echtzeit](https://go.microsoft.com/fwlink/?linkid=798760).
  
## <a name="reference-tools"></a>Referenztools
<a name="bkmk_ReferenceTools"> </a>

Literaturverzeichnis, Inhaltsverzeichnis und Index werden im Dokument im Ansichtsmodus wie erwartet angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt. Diese können Sie löschen, aber nicht bearbeiten oder aktualisieren. Erweiterte Referenztools, z. B. zum Erstellen eines Literaturverzeichnisses, eines Inhaltsverzeichnisses, eines Index oder eines Rechtsgrundlagenverzeichnisses, ind nur in der Word-Desktop-App verfügbar.
  
## <a name="rendering-of-text-boxesshapes"></a>Darstellen von Textfeldern/Formen
<a name="bkmk_textboxes"> </a>

Sie können Textfelder und Formen einfügen und dann schnell die Farbe des Texts, die Füllfarbe oder die Farbe des Rahmens ändern.

<a name="BKMK_Researcher"> </a>

Der Forscher hilft Ihnen, zuverlässige Quellen und Inhalte für Ihr Dokument zu finden und zu integrieren. Der Forscher verwendet Bing, um den entsprechenden Inhalt aus dem Internet zu ziehen. Weitere Informationen finden Sie unter [Recherchieren für Ihr Papier ganz einfach in Word](https://go.microsoft.com/fwlink/?linkid=839490).
  
## <a name="rich-media"></a>Rich-Media
<a name="bkmk_RichMedia"> </a>

Nicht verfügbar in Word für das Internet. Erweiterte Rich-Media-Features, z. B. zum Einfügen von Onlinevideos oder zum Einfügen eines PowerPoint-Videos in Ihr Word-Dokument, sind nur in der Word-Desktop-App verfügbar.

Erweiterte Tabellenformatierungen, wie das Auswählen von vordefinierten Tabellenformatvorlagen, das Festlegen von Optionen für Kopfzeilen und-Spalten sowie das Festlegen von Optionen für Schattierung und Rahmen, stehen sowohl in Word für das Internet als auch in der Word-Desktop-App zur Verfügung. Weitere Informationen zur Tabellenformatierung in Word für das Internet finden Sie unter [What es New in Word for the Internet](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="save-as-and-download-a-copy"></a>"Speichern unter" und "Kopie herunterladen"
<a name="bkmk_SaveAsDownload"> </a>

Mit Word für das Internet können Sie eine Kopie des Dokuments lokal speichern.
  
## <a name="save-as-pdf"></a>Als PDF speichern
<a name="bkmk_pdf"> </a>

Sie können Word für das Internet verwenden, um Ihre Dateien in PDFs zu speichern oder zu konvertieren, damit Sie Sie freigeben oder mit kommerziellen Druckern ausdrucken können. Sie benötigen keine weitere Software und keine Add-Ins.
  
## <a name="share"></a>Freigeben
<a name="bkmk_Share"> </a>

Bei Word für das Internet handelt es sich um einen Vorgang mit einem Mausklick zum Senden eines Links, mit dem andere Benutzer das Dokument anzeigen oder bearbeiten können. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="simple-table-creation"></a>Einfache Tabellenerstellung
<a name="bkmk_Table"> </a>

Mit Word für das Internet können Sie eine Tabelle einfügen, Tabellentext bearbeiten und einfache Tabellenstrukturen wie das Hinzufügen oder Löschen von Zeilen und Spalten ganz einfach bearbeiten. Erweiterte Tabellenfeatures, z. B. für benutzerdefinierte Tabellenformatvorlagen, die Zellengröße, die Textrichtung und die Sortierreihenfolge, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="smartart"></a>SmartArt
<a name="bkmk_SmartArteditreformat"> </a>

Bei Word für das Internet werden vorhandene Dokumente mit SmartArt im Ansichtsmodus ordnungsgemäß angezeigt. Im Bearbeitungsmodus werden Sie als Platzhalter angezeigt, die Sie löschen, jedoch nicht bearbeiten können. Sie können nicht in Word für das Internet verschoben oder geändert werden. Erweiterte Grafikfeatures wie das Einfügen von Shapes, Diagrammen, Textfeldern, SmartArt oder WordArt sind nur in der Word-Desktop-App verfügbar.
  
## <a name="table-of-contents"></a>Inhalt
<a name="bkmk_TOC"> </a>

Nicht verfügbar in Word für das Internet. Sie können ein vorhandenes Inhaltsverzeichnis anzeigen, aber nicht mit Word für das Internet erstellen. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie Ihrem Dokument in Word ein Inhaltsverzeichnis hinzufügen. Erfahren Sie mehr über das [Erstellen eines Inhaltsverzeichnisses](https://go.microsoft.com/fwlink/p/?LinkId=324414). 
  
## <a name="table-of-authority"></a>Rechtsgrundlagenverzeichnis
<a name="bkmk_TableofAuthority"> </a>

Nicht verfügbar in Word für das Internet. Sie können eine vorhandene Berechtigungstabelle anzeigen, jedoch nicht mit Word für das Internet erstellen.
  
## <a name="tap"></a>Tippen
<a name="BKMK_Tap"> </a>

Verwenden Sie das Feature zum Tippen, um Inhalte zu suchen und wiederzuverwenden. Tippen Sie auf relevante Dateien, die Sie am häufigsten verwenden, sodass Sie sich auf die Dokumenterstellung und nicht auf das Suchen nach Dateien und Informationen konzentrieren können. Weitere Informationen finden Sie unter [Suchen und Verwenden von benötigten Inhalten](https://go.microsoft.com/fwlink/?linkid=825878).
  
## <a name="tell-me"></a>Sie wünschen...
<a name="BKMK_Tell_Me_Word"> </a>

Wenn Sie etwas in Word für das Internet durchführen müssen, aber nicht wissen, wie, können Sie die Funktion "mich suchen" verwenden, um schnell zu finden, wonach Sie suchen. Die Funktion versteht, welche Aktion Sie auszuführen versuchen, und hilft Ihnen mithilfe von Vorschlägen, dies schneller zu tun.
  
## <a name="translation-service"></a>Übersetzungsdienst
<a name="bkmk_translation"> </a>

Sie können Text, ob eine ganze Datei, ausgewählte Wörter für Ausdrücke oder einzelne Wörter, mit den in Word für das Internet verfügbaren Übersetzungstools übersetzen. 
  
## <a name="undo-and-redo"></a>Rückgängigmachen und Wiederholen
<a name="bkmk_Undoredo"> </a>

Sie können eine Reihe von Aktionen rückgängig machen oder eine Aktion wiederholen.
  
## <a name="vba-and-forms-scripts"></a>VBA- und Formularskripts
<a name="bkmk_VBAforms"> </a>

ActiveX-Steuerelemente, eingebettete OLR-Objekte und eine Signatur Leiste werden im Dokument wie erwartet im Ansichtsmodus angezeigt. Im Bearbeitungsmodus werden Sie als Platzhalter angezeigt, die Sie löschen, jedoch nicht bearbeiten können. Sie können nicht in Word für das Internet verschoben oder geändert werden. Erweiterte Steuerelemente und Makros sind nur in der Word-Desktop-App verfügbar. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="view-and-add-comments"></a>Anzeigen und Hinzufügen von Kommentaren
<a name="bkmk_Viewaddcomments"> </a>

Mit Word für das Internet können Sie Kommentare im Bearbeitungsmodus einfügen, bearbeiten oder löschen. Sie können außerdem auf einen Kommentar antworten oder ihn als erledigt kennzeichnen, wie Sie es auch in Word auf dem Desktop tun würden.
  
## <a name="watermarks"></a>Wasserzeichen
<a name="bkmk_watermarks"> </a>

Bei Wasserzeichen handelt es sich um Text oder Bilder, die hinter dem Text in Ihrem Dokument platziert werden. Sie können z. B. Wasserzeichen für Text hinzufügen, z. B. „Entwurf" oder „Vertraulich". Word enthält eine Sammlung von Wasserzeichen, oder Sie können Ihr eigenes benutzerdefiniertes Wasserzeichen erstellen, z. B. ein Unternehmenslogo für Ihr Dokument. 
  
## <a name="word-count"></a>Wörter zählen
<a name="bkmk_Wordcount"> </a>

Word für das Internet gibt Ihnen eine grobe Wortanzahl für Wörter in Ihrem Dokument. Erfahren Sie mehr über [die Wortanzahl in Word für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=271864).

<a name="bkmk_Zoom"> </a>

Die Zoomfunktion ist im Ansichtsmodus verfügbar. Verwenden Sie in Word für das Internet die Ansichtseinstellungen Ihres Browsers, um im Bearbeitungsmodus ein-oder auszuzoomen. Erfahren Sie mehr über [die Wortanzahl in Word für das Internet](https://go.microsoft.com/fwlink/p/?LinkId=271864).
