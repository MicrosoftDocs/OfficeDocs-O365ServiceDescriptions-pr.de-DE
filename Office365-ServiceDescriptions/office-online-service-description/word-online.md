# <a name="word-online"></a>Word Online

Word Online (ehemals Word Web App) erweitert Ihre Microsoft Word-Benutzeroberfläche in den Webbrowser, in dem Sie Dokumente direkt auf der Website bearbeiten können, auf der das Dokument gespeichert ist. 
  
## <a name="alignment-guides-and-live-layout"></a>Hilfslinien und Echtzeitlayout
<a name="bkmk_AlignmentGuidesLivelayout"> </a>

Lineale und Gitternetzlinie sind in Word Online nicht verfügbar. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="apply-styles"></a>Anwenden von Formatvorlagen
<a name="bkmk_Styles"> </a>

Die Absatz- und Zeichenformatvorlagen, die in einem Dokument gespeichert sind, können in Word Online auf Text angewendet werden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="office-add-ins"></a>Office-Add-Ins
<a name="bkmk_AppsforOffice"> </a>

Sie können im Office Store Add-Ins für Word erwerben. Office-Add-Ins können nur in der Word-Desktop-App verwendet werden und sind in Word Online nicht verfügbar. Informationen zum Abrufen eines Add-Ins für Word finden Sie [hier](https://go.microsoft.com/fwlink/?LinkId=391443).
  
## <a name="autosave"></a>Autosave
<a name="bkmk_Save"> </a>

Word Online speichert Ihr Dokument automatisch, wenn Sie Änderungen vornehmen.
  
## <a name="autotext"></a>AutoText
<a name="bkmk_autotext"> </a>

Mithilfe von AutoText können Sie aus dem AutoText-Katalog Bausteine aus vorformatiertem Text einfügen. Dies ist beispielsweise sinnvoll, wenn Sie denselben Textblock wiederholt eingeben müssen und der Text viel Formatierung enthält. AutoText-Einträge werden durch Hinzufügen von ausgewähltem Text zum AutoText-Katalog erstellt. Weitere Informationen hierzu finden Sie unter [Automatisches Einfügen von Text](https://go.microsoft.com/fwlink/?linkid=846739).
  
## <a name="browse-and-start-from-professionally-designed-templates"></a>Suchen nach und Verwenden von professionell gestalteten Vorlagen
<a name="bkmk_DesignedTemplates"> </a>

Nicht verfügbar in Word Online. Erweiterte Designfeatures, z. B. zum Erstellen von Dokumenten auf Grundlage einer der zahlreichen professionell gestalteten Vorlagen von Office.com, sind nur in der Word-Desktop-App verfügbar. Office 365-Kunden können eine Dokumentbibliothek allerdings so konfigurieren, dass eine benutzerdefinierte Vorlage gestartet wird, wenn ein Benutzer ein neues Dokument erstellt. Weitere Informationen finden Sie unter [Hinzufügen eines Inhaltstyps zu einer Liste oder Bibliothek](https://go.microsoft.com/fwlink/p/?LinkId=391446).
  
## <a name="bullets-and-numbering"></a>Aufzählung und Nummerierung
<a name="bkmk_BulletsNumbering"> </a>

In Word Online haben Sie die Wahl zwischen drei Formatvorlagen für Aufzählungszeichen und fünf Formatvorlagen für Nummerierungen. Klicken Sie auf die Schaltflächen "Einzug vergrößern" und "Einzug verkleinern", um die Listenebene für vorhandene Aufzählungen und nummerierte Listen im Dokument sowie für in der Word-Desktop-App erstellte Aufzählungen und Nummerierungen zu ändern. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="captions"></a>Beschriftungen
<a name="bkmk_Captions"> </a>

Nicht verfügbar in Word Online. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie einem Dokument in Word Beschriftungen hinzuzufügen. Erfahren Sie mehr über das [Hinzufügen von Beschriftungen in Word 2013](https://go.microsoft.com/fwlink/p/?LinkId=282312).
  
## <a name="citations-and-bibliography"></a>Zitate und Literaturverzeichnis
<a name="bkmk_Citations"> </a>

Nicht verfügbar in Word Online. Sie können vorhandene Zitate in Word Online anzeigen, aber keine neuen Zitate hinzufügen. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie Ihrem Dokument in Word ein Literaturverzeichnis und Zitate hinzufügen. Erfahren Sie mehr über das [Erstellen eines Literaturverzeichnisses in Word 2013](https://go.microsoft.com/fwlink/p/?LinkId=282314). 
  
## <a name="clipboard"></a>Zwischenablage
<a name="bkmk_Clipboard"> </a>

Aufgrund technologischer Einschränkungen in Webbrowsern gibt es Unterschiede zwischen dem Kopieren und Einfügen von Text in Office Online und dem Kopieren und Einfügen von Text in den Office-Desktopanwendungen. Erfahren Sie mehr über das [Kopieren und Einfügen in Office Online](https://go.microsoft.com/fwlink/p/?LinkId=282321).
  
## <a name="collaboration--advanced"></a>Erweiterte Zusammenarbeit
<a name="bkmk_AdvancedCollaboration"> </a>

Nicht verfügbar in Word Online. Erweiterte Features für die Zusammenarbeit, z. B. zum Aktivieren der Nachverfolgung von Änderungen und das Zusammenführen, Vergleichen und Zusammenfassen von Dokumenten, sind nur in der Word-Desktop-App verfügbar. Überarbeitungen werden in Word Online im Ansichtsmodus wie erwartet angezeigt.
  
## <a name="content-controls"></a>Inhaltssteuerelemente
<a name="bkmk_contentcontrols"> </a>

Inhaltssteuerelemente sind einzelne Steuerelemente, die Sie für die Verwendung in Vorlagen, Formularen und Dokumenten hinzufügen und anpassen können. Weitere Informationen zu Inhaltssteuerelementen finden Sie unter [Inhaltssteuerelemente](https://go.microsoft.com/fwlink/p/?LinkId=846741).
  
## <a name="cover-pages"></a>Deckblätter
<a name="bkmk_coverpage"> </a>

Microsoft Word bietet einen Katalog von vorgefertigten Deckblättern. Sie können einfach ein Deckblatt auswählen und den Beispieltext durch Ihren eigenen Text ersetzen.
  
## <a name="creation-of-advanced-document-elements"></a>Erstellen erweiterter Dokumentelemente
<a name="bkmk_Creationadvanceddocumentelements"> </a>

Nicht verfügbar in Word Online. Erweiterte Dokumentelemente wie AutoText, Deckblätter, Literaturverzeichnis, Inhaltsverzeichnis, Index, Formeln sowie Wasserzeichen sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="document-parts-and-themes"></a>Dokumentbausteine und Designs
<a name="bkmk_Documentpartsandthemes"> </a>

Nicht verfügbar in Word Online. Dokumentbausteine und Designs wie Inhaltssteuerelemente und vorformatierte Sammlungen aus Farbschema, Schriftartenschema und Voreinstellungen für grafische Effekte sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="document-translation"></a>Übersetzung von Dokumenten
<a name="bkmk_DocumentTranslation"> </a>

Übersetzung von Dokumenten direkt im Browser. Sogar Dokumentkommentare werden übersetzt.
  
## <a name="dropbox"></a>Dropbox
<a name="bkmk_Dropbox"> </a>

Bei Dropbox handelt es sich um einen Dateihostingdienst, der einen Cloudspeicher, eine Dateisynchronisierung, eine persönliche Cloud und Clientsoftware bietet.
  
## <a name="equations"></a>Formeln
<a name="bkmk_equations"> </a>

Microsoft Word enthält integrierte Unterstützung zum Schreiben und Bearbeiten von Formeln sowie Gleichungen, die Sie in Ihren Dokumenten ablegen können.
  
## <a name="find-and-replace"></a>Suchen und Ersetzen
<a name="bkmk_Find"> </a>

Suchen und ersetzen Sie Text in Ihrem Dokument mithilfe des Tools zum Suchen und Ersetzen in Word Online und der Word-Desktop-App. Weitere Informationen zur Verwendung von Suchen und Ersetzen in Word Online finden Sie unter [Neuerungen in Word Online](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="font-formatting"></a>Schriftartformatierung
<a name="bkmk_Fontformatting"> </a>

Mit Word Online können Sie Schriftarten, Schriftgrade und verschiedene Attribute für die Schriftartformatierung anwenden, u. a. Fett- und Kursivdruck, Unterstreichungen und Hochstellungen. Markieren Sie Text, ändern Sie die Schriftfarbe, und löschen Sie die Formatierung für markierten Text. Zum Formatieren von Text in Word Online ist die Dokumentdesignformatierung verfügbar. Die aktiven Schriftartendesigns finden Sie oben im Schriftartenkatalog, die Farbdesigns im Schriftfarbenkatalog. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="footnotes-and-endnotes"></a>Fußnoten und Endnoten
<a name="bkmk_Footnotes"> </a>

Sie können Ihren Dokumenten in Word Online Fußnoten und Endnoten hinzufügen. Erfahren Sie mehr über das [Hinzufügen von Fußnoten und Endnoten in Word Online](https://go.microsoft.com/fwlink/p/?LinkId=397306).
  
## <a name="full-fidelity-reading-view"></a>Originalgetreue Leseansicht
<a name="bkmk_FullFidelityReadingView"> </a>

Wenn Sie Ihr Dokument in OneDrive oder SharePoint öffnen, öffnet Word Online das Dokument im Browser. Layout und Formatierung entsprechen dem, was Sie sehen, wenn Sie das Dokument in der Word-Desktop-App in der Drucklayoutansicht öffnen.
  
## <a name="graphic-effect-presets"></a>Voreinstellungen für grafische Effekte
<a name="bkmk_presets"> </a>

Sie können Voreinstellungen verwenden, um Spezialeffekte, wie z. B. Drehung, Schatten, Sättigung und Texteffekte, auf Grafiken anzuwenden.
  
## <a name="headers-and-footers"></a>Kopf- und Fußzeilen
<a name="bkmk_WordHeadersandFooters"> </a>

Sie können sowohl in Word Online als auch in der Word-Desktop-App eine Kopfzeile und eine Fußzeile in Dokumente einfügen. Sie haben auch die Möglichkeit, Kopf- und Fußzeilen auf alle Seiten außer die erste Seite eines Dokuments anzuwenden. Weitere Informationen zur Verwendung von Kopf- und Fußzeilen in Word Online finden Sie unter [Neuerungen in Word Online](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="hyperlinks"></a>Hyperlinks
<a name="bkmk_Hyperlinks"> </a>

Mit Word Online können Sie ganz einfach Links zu Lesezeichen in einem Dokument erstellen und eine QuickInfo festlegen, die angezeigt wird, wenn der Cursor auf den Link zeigt. Erfahren Sie mehr über [Links in Word Online](https://go.microsoft.com/fwlink/p/?LinkId=271863).
  
## <a name="index"></a>Index 
<a name="bkmk_index"> </a>

Ein Index enthält die Begriffe und Themen, die in einem Dokument erläutert werden, zusammen mit den Seiten, auf denen diese angezeigt werden. Um einen Index zu erstellen, markieren Sie die Indexeinträge, indem Sie den Namen des Haupteintrags und den Querverweis in Ihrem Dokument bereitstellen und dann den Index erstellen.
  
## <a name="ink-editor"></a>Freihand-Editor
<a name="BKMK_InkEditor"> </a>

Mit dem frei Hand Editor können Sie Dokumente mit natürlichen Gesten wie Kreisen bearbeiten und formatieren, um Text auszuwählen und um Text zu löschen.
  
## <a name="insert-online-pictures"></a>Einfügen von Onlinebildern
<a name="bkmk_InsertPicturessClipArt"> </a>

Mit Word Online können Sie lokal auf der Festplatte gespeicherte Bilder oder Bing-Bilder einfügen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="irm-and-password-protection"></a>IRM- und Kennwortschutz
<a name="bkmk_IRMpasswoprdprotect"> </a>

Word Online zeigt Dokumente an, die mit der Verwaltung von Informationsrechten geschützt sind (Information Rights Management, IRM). Diese Dokumente können allerdings nicht im Browser bearbeitet werden, und Sie können IRM-geschützte Dokumente nicht in Word Online erstellen. Word Online kann Dokumente nicht öffnen, die mit einem Kennwort verschlüsselt sind. Erweiterte Funktionen für den Dokumentschutz, z. B. das Erstellen IRM-geschützter Dokumente und das Anwenden von Kennwortschutz, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="learning-tools"></a>Lerntools
<a name="BKMK_LearningTools"> </a>

Lern Tools sollen Ihnen dabei helfen, ihre Lesekompetenz zu verbessern, indem Sie Ihre Fähigkeit erhöhen, Wörter richtig auszusprechen, schnell und genau zu lesen und zu verstehen, was Sie lesen.

Beim Speichern wird das Onlinedokument mit Ihren Änderungen aktualisiert. Sie können auch eine Offlinekopie des Dokuments herunterladen und speichern; diese wird aber nicht mit der Onlineversion synchronisiert sein. Erfahren Sie mehr über [Office Web Apps Server](https://go.microsoft.com/fwlink/p/?LinkId=282318).

## <a name="proofing-tools"></a>Korrekturhilfen
<a name="bkmk_ProofingTools"> </a>

Word Online überprüft während der Benutzereingabe automatisch die Rechtschreibung und kennzeichnet falsch geschriebene Wörter mit einer roten Wellenlinie. Allgemeine AutoKorrektur-Aktionen sind enthalten, z. B. die Korrektur häufig vorkommender Rechtschreibfehler oder die Konvertierung von Zeichen in Symbole. Außerdem können Sie die Sprache für die Korrekturhilfe ändern oder die Rechtschreibprüfung für markierten Text deaktivieren. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="proofing-tools--advanced"></a>Erweitere Korrekturhilfen
<a name="bkmk_AdvancedProofingTools"> </a>

Nicht verfügbar in Word Online. Erweitere Korrekturhilfen, z. B. zum Hinzufügen neuer Wörter zum Wörterbuch für die Rechtschreibprüfung, zum Anwenden eines benutzerdefinierten Wörterbuchs sowie Sprach- und Übersetzungstools, sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="real-time-co-authoring"></a>Gemeinsame Dokumenterstellung in Echtzeit
<a name="bkmk_Coauthoring"> </a>

Mehrere Personen können gleichzeitig in einem Dokument arbeiten. In Word Online und Word 2016 hilft Ihnen die Echt Zeit Präsenz dabei, zu sehen, wo Ihre Mitverfasser im Dokument arbeiten, damit Sie beim Bearbeiten keine Konflikte erstellen, und Sie können Änderungen sehen, während Sie vorgenommen werden. Word 2013 unterstützt Gleichzeitiges Bearbeiten, aber es gibt keine Anwesenheitsanzeige, und Änderungen können von mehreren Autoren erst angezeigt werden, wenn das Dokument gespeichert ist. Weitere Informationen zur gemeinsamen Erstellung in Echtzeit finden Sie unter [What es New in Word Online](https://go.microsoft.com/fwlink/?LinkId=389520) und Zusammenarbeit [an Word-Dokumenten mit der gemeinsamen Dokumenterstellung in Echtzeit](https://go.microsoft.com/fwlink/?linkid=798760).
  
## <a name="reference-tools"></a>Referenztools
<a name="bkmk_ReferenceTools"> </a>

Literaturverzeichnis, Inhaltsverzeichnis und Index werden im Dokument im Ansichtsmodus wie erwartet angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt. Diese können Sie löschen, aber nicht bearbeiten oder aktualisieren. Erweiterte Referenztools, z. B. zum Erstellen eines Literaturverzeichnisses, eines Inhaltsverzeichnisses, eines Index oder eines Rechtsgrundlagenverzeichnisses, ind nur in der Word-Desktop-App verfügbar.

Mit Word Online können Sie eine Tabelle einfügen, Tabellentext bearbeiten und die grundlegende Tabellenstruktur einfach bearbeiten, beispielsweise durch Hinzufügen oder Löschen von Zeilen und Spalten. Erweiterte Tabellenfeatures, z. B. für benutzerdefinierte Tabellenformatvorlagen, die Zellengröße, die Textrichtung und die Sortierreihenfolge, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="smartart"></a>SmartArt
<a name="bkmk_SmartArteditreformat"> </a>

Mit Word Online werden vorhandene Dokumente mit SmartArt-Elementen im Ansichtsmodus richtig angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt. Diese können Sie löschen, aber nicht bearbeiten. Es ist in Word Online nicht möglich, sie zu verschieben oder ihre Größe zu ändern. Erweiterte Gestaltungsfunktionen, z. B. zum Einfügen von Formen, Diagrammen, Textfeldern, SmartArt- oder WordArt-Elementen, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="table-of-contents"></a>Inhalt
<a name="bkmk_TOC"> </a>

Nicht verfügbar in Word Online. Mit Word Online können Sie bereits vorhandene Inhaltsverzeichnisse anzeigen, aber keine Inhaltsverzeichnisse erstellen. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie Ihrem Dokument in Word ein Inhaltsverzeichnis hinzufügen. Erfahren Sie mehr über das [Erstellen eines Inhaltsverzeichnisses](https://go.microsoft.com/fwlink/p/?LinkId=324414). 
  
## <a name="table-of-authority"></a>Rechtsgrundlagenverzeichnis
<a name="bkmk_TableofAuthority"> </a>

Nicht verfügbar in Word Online. Mit Word Online können Sie ein vorhandenes Rechtsgrundlagenverzeichnis anzeigen, aber Sie können kein neues erstellen.
  
## <a name="tap"></a>Tippen
<a name="BKMK_Tap"> </a>

Verwenden Sie das Feature zum Tippen, um Inhalte zu suchen und wiederzuverwenden. Tippen Sie auf relevante Dateien, die Sie am häufigsten verwenden, sodass Sie sich auf die Dokumenterstellung und nicht auf das Suchen nach Dateien und Informationen konzentrieren können. Weitere Informationen finden Sie unter [Suchen und Verwenden von benötigten Inhalten](https://go.microsoft.com/fwlink/?linkid=825878).
  
## <a name="tell-me"></a>Funktion "Was möchten Sie tun?"
<a name="BKMK_Tell_Me_Word"> </a>

Wenn Sie eine Aufgabe in Word Online ausführen möchten, aber nicht weiter wissen, können Sie die Suchfunktion "Was möchten Sie tun?" verwenden, um das Gesuchte schnell zu finden. Die Funktion versteht, welche Aktion Sie auszuführen versuchen, und hilft Ihnen mithilfe von Vorschlägen, dies schneller zu tun.
  
## <a name="translation-service"></a>Übersetzungsdienst
<a name="bkmk_translation"> </a>

Sie können Text (eine ganze Datei, ausgewählte Wörter für Ausdrücke oder einzelne Wörter) mit den in Word Online verfügbaren Übersetzungstools übersetzen. 
  
## <a name="undo-and-redo"></a>Rückgängigmachen und Wiederholen
<a name="bkmk_Undoredo"> </a>

Sie können eine Reihe von Aktionen rückgängig machen oder eine Aktion wiederholen.
  
## <a name="vba-and-forms-scripts"></a>VBA- und Formularskripts
<a name="bkmk_VBAforms"> </a>

ActiveX-Steuerelemente, eingebettete OLR-Objekte und Signaturzeilen werden in Dokumenten im Ansichtsmodus wie erwartet angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt. Diese können Sie löschen, aber nicht bearbeiten. Es ist in Word Online nicht möglich, sie zu verschieben oder ihre Größe zu ändern. Erweiterte Steuerelemente und Makros sind nur in der Word-Desktop-App verfügbar. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="view-and-add-comments"></a>Anzeigen und Hinzufügen von Kommentaren
<a name="bkmk_Viewaddcomments"> </a>

Mit Word Online können Sie Kommentare im Bearbeitungsmodus einfügen, bearbeiten oder löschen. Sie können außerdem auf einen Kommentar antworten oder ihn als erledigt kennzeichnen, wie Sie es auch in Word auf dem Desktop tun würden.
  
## <a name="watermarks"></a>Wasserzeichen
<a name="bkmk_watermarks"> </a>

Bei Wasserzeichen handelt es sich um Text oder Bilder, die hinter dem Text in Ihrem Dokument platziert werden. Sie können z. B. Wasserzeichen für Text hinzufügen, z. B. „Entwurf" oder „Vertraulich". Word enthält eine Sammlung von Wasserzeichen, oder Sie können Ihr eigenes benutzerdefiniertes Wasserzeichen erstellen, z. B. ein Unternehmenslogo für Ihr Dokument.

<a name="bkmk_Zoom"> </a>

Erfahren Sie mehr über das [Zählen von Wörtern in Word Online](https://go.microsoft.com/fwlink/p/?LinkId=271864).
