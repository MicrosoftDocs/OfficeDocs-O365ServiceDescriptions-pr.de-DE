# <a name="word-for-the-web"></a>Word für das Web

Word für das Web (früher Word Web App) erweitert Ihre Microsoft Word-Erfahrung auf den Webbrowser, in dem Sie mit Dokumenten direkt auf der Website arbeiten können, auf der das Dokument gespeichert ist. 
  
## <a name="alignment-guides-and-live-layout"></a>Hilfslinien und Echtzeitlayout

Lineale und Gitternetzlinien sind in Word für das Web nicht verfügbar. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="apply-styles"></a>Anwenden von Formatvorlagen

Die absatz- und zeichenformatvorlagen, die in einem Dokument gespeichert werden, können auf Text in Word für das Web angewendet werden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="office-add-ins"></a>Office-Add-Ins

Sie können im Office Store Add-Ins für Word erwerben. Office-Add-Ins funktionieren nur in der Word-Desktop-App und sind in Word für das Web nicht verfügbar. Informationen zum Abrufen eines Add-Ins für Word finden Sie [hier](https://go.microsoft.com/fwlink/?LinkId=391443).
  
## <a name="autosave"></a>Automatisches Weben

Word für das Web speichert Ihr Dokument automatisch, wenn Sie Änderungen vornehmen.
  
## <a name="autotext"></a>AutoText

Mithilfe von AutoText können Sie aus dem AutoText-Katalog Bausteine aus vorformatiertem Text einfügen. Dies ist beispielsweise sinnvoll, wenn Sie denselben Textblock wiederholt eingeben müssen und der Text viel Formatierung enthält. AutoText-Einträge werden durch Hinzufügen von ausgewähltem Text zum AutoText-Katalog erstellt. Weitere Informationen hierzu finden Sie unter [Automatisches Einfügen von Text](https://go.microsoft.com/fwlink/?linkid=846739).
  
## <a name="browse-and-start-from-professionally-designed-templates"></a>Suchen nach und Verwenden von professionell gestalteten Vorlagen

In Word für das Web nicht verfügbar. Erweiterte Designfeatures, z. B. zum Erstellen von Dokumenten auf Grundlage einer der zahlreichen professionell gestalteten Vorlagen von Office.com, sind nur in der Word-Desktop-App verfügbar. Kunden können jedoch eine Dokumentbibliothek so konfigurieren, dass eine benutzerdefinierte Vorlage gestartet wird, wenn ein Benutzer ein neues Dokument erstellt. Weitere Informationen finden Sie unter [Hinzufügen eines Inhaltstyps zu einer Liste oder Bibliothek](https://go.microsoft.com/fwlink/p/?LinkId=391446).
  
## <a name="bullets-and-numbering"></a>Aufzählung und Nummerierung

Mit Word für das Web können Sie eine Auswahl von drei Aufzählungsformatvorlagen oder fünf Nummerierungsformatvorlagen anwenden. Klicken Sie auf  die Schaltflächen **Einzug** erhöhen und Einzug verringern, um die Listenebene für vorhandene aufzählungs- und nummerierte Listen im Dokument sowie die in der Word-Desktop-App erstellten zu ändern. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="captions"></a>Beschriftungen

In Word für das Web nicht verfügbar. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie einem Dokument in Word Beschriftungen hinzuzufügen. Erfahren Sie mehr über das [Hinzufügen von Beschriftungen in Word 2013](https://go.microsoft.com/fwlink/p/?LinkId=282312).
  
## <a name="citations-and-bibliography"></a>Zitate und Literaturverzeichnis

In Word für das Web nicht verfügbar. Sie können vorhandene Zitate in Word für das Web anzeigen, aber sie können keine Zitate hinzufügen. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie Ihrem Dokument in Word ein Literaturverzeichnis und Zitate hinzufügen. Erfahren Sie mehr über das [Erstellen eines Literaturverzeichnisses in Word 2013](https://go.microsoft.com/fwlink/p/?LinkId=282314). 
  
## <a name="clipboard"></a>Zwischenablage

Aufgrund der Einschränkungen der Webbrowsertechnologie unterscheidet sich das Kopieren und Einfügen von Text in Office für das Web vom Kopieren und Einfügen von Text in den Office-Desktopanwendungen. Erfahren Sie mehr [über kopieren und einfügen in Office für das Web](https://go.microsoft.com/fwlink/p/?LinkId=282321).
  
## <a name="collaboration-advanced"></a>Zusammenarbeit, erweitert

In Word für das Web nicht verfügbar. Erweiterte Features für die Zusammenarbeit, z. B. zum Aktivieren der Nachverfolgung von Änderungen und das Zusammenführen, Vergleichen und Zusammenfassen von Dokumenten, sind nur in der Word-Desktop-App verfügbar. Wie erwartet werden nachverfolgte Änderungen im Ansichtsmodus in Word für das Web angezeigt.
  
## <a name="content-controls"></a>Inhaltssteuerelemente

Inhaltssteuerelemente sind einzelne Steuerelemente, die Sie für die Verwendung in Vorlagen, Formularen und Dokumenten hinzufügen und anpassen können. Weitere Informationen zu Inhaltssteuerelementen finden Sie unter [Inhaltssteuerelemente](https://go.microsoft.com/fwlink/p/?LinkId=846741).
  
## <a name="cover-pages"></a>Deckblätter

Microsoft Word bietet einen Katalog von vorgefertigten Deckblättern. Sie können einfach ein Deckblatt auswählen und den Beispieltext durch Ihren eigenen Text ersetzen.
  
## <a name="creation-of-advanced-document-elements"></a>Erstellen erweiterter Dokumentelemente

In Word für das Web nicht verfügbar. Erweiterte Dokumentelemente wie AutoText, Deckblätter, Literaturverzeichnis, Inhaltsverzeichnis, Index, Formeln sowie Wasserzeichen sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="document-parts-and-themes"></a>Dokumentbausteine und Designs

In Word für das Web nicht verfügbar. Dokumentbausteine und Designs wie Inhaltssteuerelemente und vorformatierte Sammlungen aus Farbschema, Schriftartenschema und Voreinstellungen für grafische Effekte sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="document-translation"></a>Übersetzung von Dokumenten

Übersetzung von Dokumenten direkt im Browser. Sogar Dokumentkommentare werden übersetzt.
  
## <a name="dropbox"></a>Dropbox

Bei Dropbox handelt es sich um einen Dateihostingdienst, der einen Cloudspeicher, eine Dateisynchronisierung, eine persönliche Cloud und Clientsoftware bietet.
  
## <a name="equations"></a>Formeln

Microsoft Word enthält integrierte Unterstützung zum Schreiben und Bearbeiten von Formeln sowie Gleichungen, die Sie in Ihren Dokumenten ablegen können.
  
## <a name="find-and-replace"></a>Suchen und Ersetzen

Suchen und ersetzen Sie Text in Ihrem Dokument mithilfe des Tools Suchen und Ersetzen sowohl in Word für das Web als auch in der Word-Desktop-App. Weitere Informationen zur Verwendung von Suchen und Ersetzen in Word für das Web finden Sie unter [Suchen und Ersetzen von Text](https://support.office.com/article/c6728c16-469e-43cd-afe4-7708c6c779b7#ID0EAABAAA=Web).
  
## <a name="font-formatting"></a>Schriftartformatierung

Mit Word für das Web können Sie Schriftart, Schriftgrad und mehrere Schriftartformatierungsattribute anwenden, einschließlich &mdash; Fett, Italic, Unterstreichung und Hochgestellt. Hervorheben von Text, Ändern der Schriftfarbe und klare Formatierung für markierten Text. Die Formatierung des Dokumentdesigns ist zum Formatieren von Text in Word für das Web verfügbar. Suchen Sie die aktiven Designschriftarten am oberen Rand des Schriftartenkatalogs und der Designfarben im Katalog Schriftartenfarbe. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="footnotes-and-endnotes"></a>Fußnoten und Endnoten

Sie können Ihren Dokumenten in Word für das Web Fußnoten und Endnoten hinzufügen. Erfahren Sie mehr [über das Hinzufügen von Fußnoten und Endnoten in Word für das Web.](https://go.microsoft.com/fwlink/p/?LinkId=397306)
  
## <a name="full-fidelity-reading-view"></a>Originalgetreue Leseansicht

Wenn Sie Ihr Dokument in OneDrive oder SharePoint öffnen, öffnet Word für das Web das Dokument im Browser. Layout und Formatierung entsprechen dem, was Sie sehen, wenn Sie das Dokument in der Word-Desktop-App in der Drucklayoutansicht öffnen.
  
## <a name="graphic-effect-presets"></a>Voreinstellungen für grafische Effekte

Sie können Voreinstellungen verwenden, um Spezialeffekte, wie z. B. Drehung, Schatten, Sättigung und Texteffekte, auf Grafiken anzuwenden.
  
## <a name="headers-and-footers"></a>Kopf- und Fußzeilen

Fügen Sie eine Kopf- und Fußzeile in Ihr Dokument sowohl in Word für das Web als auch in der Word-Desktop-App ein. Sie haben auch die Möglichkeit, Kopf- und Fußzeilen auf alle Seiten außer die erste Seite eines Dokuments anzuwenden. Weitere Informationen zur Verwendung von Kopf- und Fußzeilen in Word für das Web finden Sie unter [What's New in Word for the web](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="hyperlinks"></a>Hyperlinks

Mit Word für das Web können Sie ganz einfach Links zu Lesezeichen in einem Dokument erstellen und eine Bildschirminfo festlegen, die angezeigt wird, wenn der Cursor auf den Link zeigt. Erfahren Sie mehr [über Hyperlinks in Word für das Web](https://go.microsoft.com/fwlink/p/?LinkId=271863).
  
## <a name="index"></a>Index

Ein Index enthält die Begriffe und Themen, die in einem Dokument erläutert werden, zusammen mit den Seiten, auf denen diese angezeigt werden. Um einen Index zu erstellen, markieren Sie die Indexeinträge, indem Sie den Namen des Haupteintrags und den Querverweis in Ihrem Dokument bereitstellen und dann den Index erstellen.
  
## <a name="insert-online-pictures"></a>Einfügen von Onlinebildern

Mit Word für das Web können Sie ein Bild aus einer Datei einfügen, die auf Ihrer lokalen Festplatte oder in Bing Images gespeichert ist. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="irm-and-password-protection"></a>IRM- und Kennwortschutz

Word für das Web zeigt Dokumente an, die durch die Verwaltung von Informationsrechten (Information Rights Management, IRM) geschützt sind. Diese Dokumente können jedoch nicht im Browser bearbeitet werden, und Sie können keine IRM-geschützten Dokumente in Word für das Web erstellen. Word für das Web kann keine Dokumente öffnen, die mit einem Kennwort verschlüsselt sind. Erweiterte Dokumentschutzfeatures, z. B. das Erstellen von IRM-geschützten Dokumenten und das Anwenden von Kennwortschutz, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="learning-tools"></a>Lerntools

Lerntools sollen Ihnen helfen, Ihre Lesefähigkeit zu verbessern, indem Sie Ihre Fähigkeit erhöhen, Wörter richtig aussagen zu können, schnell und genau zu lesen und zu verstehen, was Sie lesen. 
  
## <a name="lists"></a>Listen

Beginnen Sie in Word für das Web mit der Eingabe direkt unter einer vorhandenen nummerierten Liste, und Die nächste Zeile wird automatisch Teil der Liste. Erfahren Sie mehr [über das Erstellen einer Liste in Word für das Web](https://go.microsoft.com/fwlink/p/?LinkId=397307).
  
## <a name="mail-merge"></a>Seriendruck

In Word für das Web nicht verfügbar. Erweiterte Features für die Datenintegration, z. B. Seriendruck unter Verwendung einer Excel-Tabelle, sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="offline-viewing-and-authoring"></a>Offlineansicht und -erstellung

In Word für das Web nicht verfügbar. Office für das Web erfordert eine Internetverbindung und einen Internetbrowser. Sie benötigen die auf Ihrem Computer installierte Word-Desktop-App zum Anzeigen und Bearbeiten eines Dokuments, während die Verbindung mit dem Internet oder dem lokalen Office Web Apps Server Ihrer Organisation getrennt ist. Wenn Sie die Word-Desktop-App installiert haben, können Sie alle Word-Funktionen zum Anzeigen und Bearbeiten Ihres Dokuments verwenden. Beim Speichern wird das Onlinedokument mit Ihren Änderungen aktualisiert. Sie können auch eine Offlinekopie des Dokuments herunterladen und speichern; diese wird aber nicht mit der Onlineversion synchronisiert sein. Erfahren Sie mehr über [Office Web Apps Server](/webappsserver/how-office-web-apps-work-on-premises-with-sharepoint-2013).
  
## <a name="page-breaks"></a>Seitenwechsel

Einfügen von Seitenumbrüchen sowohl in Word für das Web als auch in der Word-Desktop-App.
  
## <a name="page-layout-tools"></a>Seitenlayouttools

Mit Word für das Web können Sie die Ränder, die Papiergröße und die vertikale oder horizontale Ausrichtung der Seite in Ihren Dokumenten anpassen. Word für das Web zeigt keine Lineale und Gitternetzlinien an. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859). 
  
Erfahren Sie, [wie Sie eine leere Seite in Word für das Web löschen.](https://support.office.com/article/B583020A-D26C-4661-873C-CB66FBDB03D1)
  
## <a name="page-layout-tools-advanced"></a>Seitenlayouttools, erweitert

In Word für das Web nicht verfügbar. Erweiterte Seitenlayoutfeatures wie Seitenränder und Zeilennummern sind nur in der Word-Desktop-App verfügbar.
  
## <a name="page-numbers"></a>Seitenzahlen

Fügen Sie Seitenzahlen in Ihr Dokument sowohl in Word für das Web als auch in der Word-Desktop-App ein. Sie haben auch die Möglichkeit, Seitenzahlen auf alle Seiten außer die erste Seite Ihres Dokuments anzuwenden. Weitere Informationen zur Verwendung von Seitenzahlen in Word für das Web finden Sie unter [What's new in Word for the web](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="paragraph-formatting"></a>Absatzformatierung

Mit Word für das Web können Sie Absätze links, rechts oder zentriert ausrichten. Text von rechts nach links ausführen; Erhöhen oder Verringern des Einzugs; und Formatieren von Absätzen als aufzählungs- oder nummerierte Liste. Sie können die Formatierung auch löschen. Der Zeilenabstand in Word für das Web kann nicht angepasst werden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="picture-tools"></a>Bildtools

Mit Word für das Web können Sie die Größe von Bildern ändern und in Ihrem Dokument alternativen Text hinzufügen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="preformatted-font-and-color-schemes"></a>Vorformatierte Schriftarten-und Farbschemas

Sie können Designs verwenden, um den Prozess des Erstellens ansprechender, professionell aussehender Dokumente zu vereinfachen. Designs können auf Tabellen, Diagramme, Formen und Diagramme angewendet werden, sodass einheitliche Schriften, Effekte und Farbschemas entstehen.
  
## <a name="present-online"></a>Onlinepräsentation

In Word für das Web nicht verfügbar. Mit der Word 2013-Desktop-App können Sie den Office-Präsentationsdienst oder Skype for Business Online verwenden, um ein Dokument während einer Onlinebesprechung anzuzeigen, sodass Ihre Kollegen einem Link folgen können, um zu sehen, wie Sie durch das Dokument blättern. 
  
## <a name="print-to-pdf"></a>Als PDF drucken

Mit Word für das Web erhalten Sie einen Ein-Klick-Prozess zum Generieren einer PDF und zum Anzeigen des Dialogfelds Drucken. Erfahren Sie mehr [über das Drucken eines Dokuments in Word für das Web.](https://go.microsoft.com/fwlink/p/?LinkId=271860)
  
## <a name="printing-advanced"></a>Drucken, erweitert

In Word für das Web nicht verfügbar. Erweiterte Druckfeatures, z. B. zum Drucken von Markups und Dokumenteigenschaften, sind nur in der Word-Desktop-App verfügbar 
  
## <a name="proofing-tools"></a>Korrekturhilfen

Word für das Web überprüft bei der Eingabe automatisch die Rechtschreibung und wendet eine wellige rote Unterstreichung auf falsch geschriebenen Text an. Allgemeine AutoKorrektur-Aktionen sind enthalten, z. B. die Korrektur häufig vorkommender Rechtschreibfehler oder die Konvertierung von Zeichen in Symbole. Außerdem können Sie die Sprache für die Korrekturhilfe ändern oder die Rechtschreibprüfung für markierten Text deaktivieren. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="proofing-tools-advanced"></a>Korrekturhilfen, erweitert

In Word für das Web nicht verfügbar. Erweitere Korrekturhilfen, z. B. zum Hinzufügen neuer Wörter zum Wörterbuch für die Rechtschreibprüfung, zum Anwenden eines benutzerdefinierten Wörterbuchs sowie Sprach- und Übersetzungstools, sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="real-time-co-authoring"></a>Gemeinsame Dokumenterstellung in Echtzeit

Mehrere Personen können gleichzeitig in einem Dokument arbeiten. In Word für das Web und Word 2016 hilft Ihnen die Anwesenheit in Echtzeit zu sehen, wo Ihre Mitautoren im Dokument arbeiten, damit Sie beim Bearbeiten keine Konflikte erstellen und Änderungen sehen können, während sie vorgenommen werden. Word 2013 unterstützt die gleichzeitige Bearbeitung, es gibt jedoch keine Anwesenheitsanzeige, und Änderungen können von mehreren Autoren erst angezeigt werden, wenn das Dokument gespeichert wurde. Weitere Informationen zur gemeinsamen Dokumenterstellung in Echtzeit finden Sie unter [What's new in Word for the web](https://go.microsoft.com/fwlink/?LinkId=389520) und Collaborate on Word documents with [real-time co-authoring](https://go.microsoft.com/fwlink/?linkid=798760).
  
## <a name="reference-tools"></a>Referenztools

Literaturverzeichnis, Inhaltsverzeichnis und Index werden im Dokument im Ansichtsmodus wie erwartet angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt. Diese können Sie löschen, aber nicht bearbeiten oder aktualisieren. Erweiterte Referenztools, z. B. zum Erstellen eines Literaturverzeichnisses, eines Inhaltsverzeichnisses, eines Index oder eines Rechtsgrundlagenverzeichnisses, ind nur in der Word-Desktop-App verfügbar.
  
## <a name="rendering-of-text-boxesshapes"></a>Darstellen von Textfeldern/Formen

Sie können Textfelder und Formen einfügen und dann schnell die Farbe des Texts, die Füllfarbe oder die Farbe des Rahmens ändern.
  
## <a name="researcher"></a>Recherche

Researcher hilft Ihnen, zuverlässige Quellen und Inhalte für Ihr Dokument zu finden und zu integrieren. Der Forscher verwendet Bing, um die entsprechenden Inhalte aus dem Web zu ziehen. Weitere Informationen finden Sie unter [Research for your paper easily within Word](https://go.microsoft.com/fwlink/?linkid=839490).
  
## <a name="rich-media"></a>Rich-Media

In Word für das Web nicht verfügbar. Erweiterte Rich-Media-Features, z. B. zum Einfügen von Onlinevideos oder zum Einfügen eines PowerPoint-Videos in Ihr Word-Dokument, sind nur in der Word-Desktop-App verfügbar. 
  
## <a name="rich-table-formatting"></a>Umfassende Tabellenformatierung

Erweiterte Tabellenformatierungen, z. B. die Auswahl vordefinierter Tabellenformatvorlagen, Einstellungsoptionen für Kopfzeilen und Spalten und Einstellungsoptionen für Schattierungen und Rahmen, sind sowohl in Word für das Web als auch in der Word-Desktop-App verfügbar. Weitere Informationen zur Tabellenformatierung in Word für das Web finden Sie unter [What's new in Word for the web](https://go.microsoft.com/fwlink/?LinkId=389520).
  
## <a name="save-as-and-download-a-copy"></a>Speichern unter und Herunterladen einer Kopie

Mit Word für das Web können Sie eine Kopie des Dokuments lokal speichern.
  
## <a name="save-as-pdf"></a>Als PDF speichern

Sie können Word für das Web verwenden, um Ihre Dateien in PDFs zu speichern oder zu konvertieren, sodass Sie sie freigeben oder mit kommerziellen Druckern drucken können. Sie benötigen keine weitere Software und keine Add-Ins.
  
## <a name="share"></a>Freigeben

Bei Word für das Web ist es ein Ein-Klick-Prozess zum Senden eines Links, den andere zum Anzeigen oder Bearbeiten des Dokuments verwenden können. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="simple-table-creation"></a>Einfache Tabellenerstellung

Mit Word für das Web können Sie eine Tabelle einfügen, Tabellentext bearbeiten und einfach die grundlegende Tabellenstruktur bearbeiten, z. B. Zeilen und Spalten hinzufügen oder löschen. Erweiterte Tabellenfeatures, z. B. für benutzerdefinierte Tabellenformatvorlagen, die Zellengröße, die Textrichtung und die Sortierreihenfolge, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="smartart"></a>SmartArt

Mit Word für das Web werden vorhandene Dokumente mit SmartArt ordnungsgemäß im Ansichtsmodus angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt, die Sie löschen, aber nicht bearbeiten können. Sie können in Word für das Web nicht verschoben oder geändert werden. Erweiterte Grafikfeatures, z. B. das Einfügen von Formen, Diagrammen, Textfelder, SmartArt oder WordArt, sind nur in der Word-Desktop-App verfügbar.
  
## <a name="table-of-contents"></a>Inhaltsverzeichnis

In Word für das Web nicht verfügbar. Sie können ein vorhandenes Inhaltsverzeichnis anzeigen, aber sie können keins mit Word für das Web erstellen. Wenn Sie auf Ihrem Computer die Word-Desktop-App installiert haben, können Sie Ihrem Dokument in Word ein Inhaltsverzeichnis hinzufügen. Erfahren Sie mehr über das [Erstellen eines Inhaltsverzeichnisses](https://go.microsoft.com/fwlink/p/?LinkId=324414). 
  
## <a name="table-of-authority"></a>Autoritätsverzeichnis

In Word für das Web nicht verfügbar. Sie können ein vorhandenes Autoritätsverzeichnis anzeigen, aber sie können keins mit Word für das Web erstellen.
  
## <a name="tap"></a>Tippen

Verwenden Sie das Feature zum Tippen, um Inhalte zu suchen und wiederzuverwenden. Tippen Sie auf relevante Dateien, die Sie am häufigsten verwenden, sodass Sie sich auf die Dokumenterstellung und nicht auf das Suchen nach Dateien und Informationen konzentrieren können. Weitere Informationen finden Sie unter [Suchen und Verwenden von benötigten Inhalten](https://go.microsoft.com/fwlink/?linkid=825878).
  
## <a name="tell-me"></a>Sie wünschen...

Wenn Sie etwas in Word für das Web ausführen müssen, aber nicht wissen, wie, können Sie das Suchfeature Tell Me verwenden, um schnell zu finden, wofür Sie suchen. Die Funktion versteht, welche Aktion Sie auszuführen versuchen, und hilft Ihnen mithilfe von Vorschlägen, dies schneller zu tun.
  
## <a name="translation-service"></a>Übersetzungsdienst

Sie können Text, ob eine ganze Datei, ausgewählte Wörter für Ausdrücke oder einzelne Wörter, mit den in Word für das Web verfügbaren Übersetzungstools übersetzen. 
  
## <a name="undo-and-redo"></a>"Rückgängig" und "Wiederholen"

Sie können eine Reihe von Aktionen rückgängig machen oder eine Aktion wiederholen.
  
## <a name="vba-and-forms-scripts"></a>VBA- und Formularskripts

Aktive X-Steuerelemente, eingebettete OLR-Objekte und eine Signaturzeile werden wie erwartet im Ansichtsmodus im Dokument angezeigt. Im Bearbeitungsmodus werden sie als Platzhalter angezeigt, die Sie löschen, aber nicht bearbeiten können. Sie können in Word für das Web nicht verschoben oder geändert werden. Erweiterte Steuerelemente und Makros sind nur in der Word-Desktop-App verfügbar. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Dokuments im Browser und in Word](https://go.microsoft.com/fwlink/p/?LinkId=271859).
  
## <a name="view-and-add-comments"></a>Anzeigen und Hinzufügen von Kommentaren

Mit Word für das Web können Sie Kommentare im Bearbeitungsmodus einfügen, bearbeiten oder löschen. Sie können außerdem auf einen Kommentar antworten oder ihn als erledigt kennzeichnen, wie Sie es auch in Word auf dem Desktop tun würden.
  
## <a name="watermarks"></a>Wasserzeichen

Bei Wasserzeichen handelt es sich um Text oder Bilder, die hinter dem Text in Ihrem Dokument platziert werden. Sie können z. B. Wasserzeichen für Text hinzufügen, z. B. „Entwurf“ oder „Vertraulich“. Word enthält eine Sammlung von Wasserzeichen, oder Sie können Ihr eigenes benutzerdefiniertes Wasserzeichen erstellen, z. B. ein Unternehmenslogo für Ihr Dokument.  
  
## <a name="word-count"></a>Wörter zählen

Word für das Web bietet eine grobe Wortanzahl für Wörter in Ihrem Dokument. Erfahren Sie mehr über [die Wortanzahl in Word für das Web.](https://go.microsoft.com/fwlink/p/?LinkId=271864)
  
## <a name="zoom"></a>Zoom

Die Zoomfunktion ist im Ansichtsmodus verfügbar. Verwenden Sie mit Word für das Web die Ansichtseinstellungen Ihres Browsers, um im Bearbeitungsmodus zu vergrößern oder zu verkleinern. Erfahren Sie mehr über [die Wortanzahl in Word für das Web.](https://go.microsoft.com/fwlink/p/?LinkId=271864)
