# <a name="powerpoint-for-the-web"></a>PowerPoint für das Web

PowerPoint für das Web (früher PowerPoint Web App) erweitert Ihre Microsoft PowerPoint-Erfahrung auf den Webbrowser, in dem Sie mit Präsentationen direkt auf der Website arbeiten können, auf der die Präsentation gespeichert ist. Microsoft 365 Kunden mit Office für das Web können Dateien unterwegs anzeigen, erstellen und bearbeiten.
  
Der PowerPoint-Editor ist eine Web-Front-End-Komponente, die eine browserbasierte Bearbeitungsoberfläche erstellt, mit der Benutzer an Dokumenten arbeiten können, ohne die Genauigkeit zu verlieren.
  
Wenn Sie die Schaltfläche **In PowerPoint** öffnen auf der PowerPoint für die Websymbolleiste auswählen, wird die Präsentation in der PowerPoint-Desktop-App geöffnet (wenn Microsoft PowerPoint 2010 oder höher auf dem Computer installiert ist).
  
Erfahren Sie, wie Sie Office [mit Microsoft 365 business auf einem PC herunterladen und installieren.](https://support.office.com/article/72977511-dfd1-4d8b-856f-405cfb76839c)
  
## <a name="alignment-bullets-numbers"></a>Alignment, bullets, numbers

Mit PowerPoint für das Web können Sie die Absatzausrichtung ändern, Aufzählungszeichen oder Nummerierung anwenden und die Ebene des aufzählungs- oder nummerierten Texts ändern. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-basic-transitions-and-animations"></a>Apply basic transitions and animations

Treffen Sie Ihre Auswahl aus einem Katalog von Animations- und Übergangseffekten. PowerPoint für das Web unterstützt acht Übergänge und 37 Animationen. Zusätzliche Animationen und Übergänge, die nicht von PowerPoint für das Web unterstützt werden, werden in der Präsentation beibehalten und in der Bildschirmpräsentation angezeigt, können jedoch nicht in PowerPoint für das Web geändert werden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>Apply rich formatting to text, shapes, and pictures

Nicht verfügbar in PowerPoint für das Web. Erweiterte Formatierungsfeatures, z. B. mehr Farben, Farbverläufe, Eyedropper, Effekte und Formatvorlagen, sind in der PowerPoint verfügbar. Mit PowerPoint für das Web können Sie ein Textfeld hinzufügen oder aus einem Katalog von Formen auswählen und Formatvorlagen anwenden, die Füll-, Gliederungs- und Schatteneffekte definieren. 
  
## <a name="apply-themes-and-theme-variants"></a>Apply themes and theme variants

Wählen Sie aus einem Katalog mit integrierten Designs und Varianten aus, wenn Sie der Präsentation eine neue Folie hinzufügen. Designs, die nicht von PowerPoint für das Web unterstützt werden, werden in der Präsentation beibehalten, aber die Möglichkeit zum Ändern von Designs ist in PowerPoint web nicht verfügbar. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="office-add-ins"></a>Office-Add-Ins

PowerPoint für das Web unterstützt nur Inhalts-Add-Ins für Office. Inhalts-Add-Ins integrieren webbasierte Features als Inhalt, der eingebettet in eine Präsentation angezeigt werden kann. Weitere Informationen zu den Typen von Add-Ins für Office finden Sie in dieser [Übersicht](/previous-versions/office/office-2013-resource-kit/jj219429(v=office.15)).
  
## <a name="arrange-objects"></a>Anordnen von Objekten

Verschieben, Ändern der Größe, Drehen oder Reihenfolge von Formen und Textfelder in Ebenen, von hinten nach vorne in PowerPoint web. Sie können die Gruppierung von Formen aufheben, um sie einzeln zu bearbeiten. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="broadcast-slide-show"></a>Broadcast slide show

Die PowerPoint desktop app ist erforderlich, um eine Bildschirmpräsentation an eine Remotepublikum über PowerPoint für das Web zu übertragen. Die Betrachter können Livepräsentationen über PowerPoint für das Web ansehen, unabhängig davon, ob die PowerPoint installiert ist. Weitere Informationen finden Sie unter [Info zum Feature "Bildschirmpräsentation übertragen"](https://go.microsoft.com/fwlink/?LinkId=272765).
  
## <a name="clipboard"></a>Zwischenablage

Mit PowerPoint für das Web können Sie Inhalte in einer Präsentation ausschneiden, kopieren und einfügen. Ein Benutzer kann Text zwischen Office für die Webprogramme sowie zwischen Office für das Web und den Microsoft Office-Desktop-Apps auf dem Computer kopieren und einfügen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="create-and-manage-slides"></a>Create and manage slides

Hinzufügen, Neusortieren, Duplizieren, Ausblenden und Löschen von Folien in PowerPoint Web. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="create-custom-animation"></a>Create custom animation

PowerPoint für das Web enthält einen Katalog mit Animationseffekten. Animationen, die nicht von PowerPoint für das Web unterstützt werden, werden in der Präsentation beibehalten und in der Bildschirmpräsentation angezeigt, können jedoch nicht in PowerPoint web geändert werden. Zum Erstellen benutzerdefinierter Animationen muss auf Ihrem Computer die PowerPoint-Desktop-App installiert sein. Erfahren Sie mehr über das [Erstellen benutzerdefinierter Animationseffekte mit der PowerPoint 2013-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=282348).
  
## <a name="design-tools-advanced"></a>Designtools, erweitert

Nicht verfügbar in PowerPoint für das Web. Erweiterte Designfeatures, z. B. das Erstellen von Folienmastern und das Ändern von Layouts, sind nur in der PowerPoint verfügbar. Mit PowerPoint für das Web können Sie Ihre eigene Vorlagendatei als Grundlage für das Erstellen neuer Dateien verwenden oder aus einer vordefinierten Liste von Designs auswählen.
  
## <a name="dropbox"></a>Dropbox

Bei Dropbox handelt es sich um einen Dateihostingdienst, der einen Cloudspeicher, eine Dateisynchronisierung, eine persönliche Cloud und Clientsoftware bietet.
  
## <a name="embed-presentation-on-web-or-blog-page"></a>Embed presentation on web or blog page

Mit PowerPoint für das Web können Sie Präsentationen auf Websites oder Blogs einbetten, sodass jeder Ihre Informationen anzeigen kann, auch wenn er nicht über die Microsoft PowerPoint Desktop-App verfügen. Eingebettete Viewer zeigen Animationen, Übergänge sowie Audio- und Videoinhalte originalgetreu wie in der Leseansicht an. Erfahren Sie mehr über das [Einbetten einer PowerPoint-Präsentation in eine Web- oder Blogseite](https://go.microsoft.com/fwlink/?LinkId=272766).
  
## <a name="font-formatting"></a>Schriftartformatierung

PowerPoint für das Web können Sie Fett, Italic, Unterstreichung, Schriftart, Größe und Farbe auf Text anwenden. Mit „Format übertragen" können Sie auch das Format des gesamten Shapes kopieren.
  
## <a name="full-fidelity-reading-view"></a>Originalgetreue Leseansicht

Zeigen Sie Präsentationen so an, wie sie mit Animationen, Übergängen und &mdash; Audio/Video angezeigt werden sollten.
  
## <a name="full-ink-support"></a>Full ink support

Freihandeingaben können nicht in PowerPoint Web eingefügt werden, aber PowerPoint web zeigt sie wie erwartet an.
  
## <a name="full-selection-of-animations-and-transitions"></a>Full selection of animations and transitions

PowerPoint für das Web unterstützt acht Übergänge und 37 Animationen. Eine größere Auswahl bietet die PowerPoint-Desktop-App. 
  
## <a name="headers-and-footers"></a>Kopf- und Fußzeilen

Kopf- und Fußzeilen, einschließlich Datums- und Foliennummern, können nicht in PowerPoint für das Web eingefügt, bearbeitet oder gelöscht werden, aber PowerPoint für das Web zeigt sie wie erwartet an. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="hyperlinks"></a>Hyperlinks

Sie können Links einfügen, bearbeiten und folgen. Lesezeichenlinks funktionieren, können jedoch nicht PowerPoint Web bearbeitet werden.
  
## <a name="insert-online-video"></a>Einfügen von Onlinevideos

Video- und Audioinhalte werden in lese- und bildschirmpräsentation mit einer Dateigröße von 100 MB abspielen. Onlinevideos können aus YouTube eingefügt werden, und Mediensteuerelemente können in der Größe geändert, verschoben und in PowerPoint für das Web gelöscht werden. Um jedoch andere Audio- und Videodaten als YouTube einfügen zu können, benötigen Sie die Microsoft PowerPoint Desktop-App.
  
## <a name="integration-with-excel-for-charts"></a>Integration with Excel for charts

Nicht verfügbar in PowerPoint für das Web. Mit PowerPoint für das Web können Sie Excel Diagramme in einer vorhandenen Präsentation anzeigen, aber Sie können ein Excel-Diagramm nicht mithilfe von PowerPoint für das Web bearbeiten oder einfügen. Erfahren Sie mehr über das [Kopieren eines Excel-Diagramms in ein PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272773).
  
## <a name="navigation---slide-sorter"></a>Navigation – Foliensortiergerät

Die Foliensortieransicht, die Ihnen eine Ansicht Ihrer Folien in Miniaturansichten ermöglicht, vereinfacht das Sortieren und Organisieren ihrer Folien. Sie können die Foliensortieransicht verwenden, um Folien zu organisieren, Abschnitte hinzuzufügen und Folien in verschiedene Kategorien zu sortieren.
  
## <a name="offline-viewing-and-authoring"></a>Offlineansicht und -erstellung

PowerPoint für das Web wird über einen Webbrowser gestartet und basiert auf einer Internetverbindung. Um offline auf Präsentationen zugreifen zu können, muss auf Ihrem Computer die Microsoft PowerPoint-Desktop-App zum Anzeigen und Bearbeiten von Folien installiert sein.
  
## <a name="picture-cropping"></a>Bildzuschnitt

Verbessern Sie mit dem Tool für die Fotozuschneidung den Rahmen eines Motivs des Bilds. Klicken Sie dazu einfach auf die Zuschneideziehpunkte am Rand des Bilds, und ziehen Sie diese, bis das gewünschte Ergebnis erzielt ist.
  
## <a name="pictures"></a>Bilder

Mit PowerPoint für das Web können Sie bilder einfügen, die auf Ihrem Computer gespeichert sind, oder Bilder aus Bing einfügen. Sie können Bilder verschieben, ändern und zuschneiden und eine Reihe von Bildformatvorlagen anwenden. Komplexere Features für das Arbeiten mit Bildern, z. B. das Anwenden von Effekten, sind in PowerPoint web nicht verfügbar. Sie können keine Screenshots in PowerPoint für das Web erstellen, aber Screenshots, die sich in einer Präsentation befinden, werden als Bilder in PowerPoint für das Web angezeigt. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="present-online-through-skype-for-business-or-the-office-presentation-service"></a>Present online through Skype for Business or the Office Presentation Service

Nicht verfügbar in PowerPoint für das Web. Mit PowerPoint Desktop-App von Microsoft können Sie Ihre Präsentationen mit dem Office-Präsentationsdienst, einem kostenlosen, öffentlichen Dienst, bereitstellen, mit dem andere Personen in ihrem Webbrowser folgen können. Erfahren Sie mehr über [Office-Präsentationsdienst](https://go.microsoft.com/fwlink/?LinkId=272774).
  
## <a name="presenter-view"></a>Presenter view

Nicht verfügbar in PowerPoint für das Web. Nur die Microsoft PowerPoint-Desktop-App ermöglicht einem Referenten die Steuerung der Präsentationsübertragung, der Notizen, Anmerkungen und Zoomtools im Hintergrund. PowerPoint für das Web unterstützt diese Features nicht.
  
## <a name="print-to-pdf"></a>Als PDF drucken

Mit PowerPoint für das Web können Sie Ihre Präsentation in einem PDF-Reader drucken, in dem das Layout und die Formatierung Ihrer Folien wie erwartet gedruckt werden. Erfahren Sie mehr [über grundlegende Aufgaben, die Sie mithilfe von PowerPoint für das Web ausführen können.](https://go.microsoft.com/fwlink/?LinkId=272775)
  
## <a name="proofing-tools"></a>Korrekturhilfen

Sie können die Rechtschreibung überprüfen und die Korrekturprüfungssprache mithilfe des integrierten Wörterbuchs mit PowerPoint für das Web festlegen. Aber PowerPoint für das Web verwendet kein benutzerdefiniertes Wörterbuch und enthält keine Übersetzung oder einen Thesaurus.
  
## <a name="real-time-co-authoring"></a>Gemeinsame Dokumenterstellung in Echtzeit

Mehrere Autoren können gleichzeitig in PowerPoint web und PowerPoint 2016. Die Anwesenheit in Echtzeit hilft Ihnen, zu sehen, wo Ihre Mitautoren in der Präsentation arbeiten, damit Sie beim Bearbeiten keine Konflikte erstellen und Änderungen sehen können, während sie vorgenommen werden. PowerPoint 2013 unterstützt die gleichzeitige Bearbeitung, es gibt jedoch keine Anwesenheitsanzeige, und Änderungen können von mehreren Autoren erst angezeigt werden, wenn das Dokument gespeichert wurde. Weitere Informationen zur gemeinsamen Echtzeiterstellung in PowerPoint finden Sie unter Arbeiten an [PowerPoint Präsentationen](https://go.microsoft.com/fwlink/?linkid=844014).
  
## <a name="reviewer-tools-advanced"></a>Begutachtertools, erweitert

Nicht verfügbar in PowerPoint für das Web. Erweiterte Begutachterfeatures, z. B. Zusammenführungskonflikte und Vergleichspräsentationen, sind nur in der PowerPoint verfügbar. Mit PowerPoint für das Web können Sie Kommentare anzeigen, hinzufügen, bearbeiten oder löschen.
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Rechteverwaltung: Anwenden und Nutzen von IRM und Kennwortschutz

Nicht verfügbar in PowerPoint für das Web. PowerPoint für das Web zeigt Präsentationen an, die durch die Verwaltung von Informationsrechten (Information Rights Management, IRM) geschützt sind. Diese Präsentationen können jedoch nicht im Browser bearbeitet werden, und Sie können keine IRM-geschützten Präsentationen in PowerPoint Web erstellen. PowerPoint für das Web können keine Präsentationen öffnen, die digital signiert oder mit einem Kennwort verschlüsselt sind. 
  
## <a name="run-slide-show"></a>Run slide show

Mit PowerPoint für das Web können Sie Ihre Präsentation nur mit einem Webbrowser und einer Internetverbindung ausführen. Erfahren Sie mehr [über grundlegende Aufgaben, die Sie mithilfe von PowerPoint für das Web ausführen können.](https://go.microsoft.com/fwlink/?LinkId=272775)
  
## <a name="save-as-or-download-a-copy"></a>Speichern unter oder Herunterladen einer Kopie

PowerPoint für das Web speichert Ihre Arbeit jedes Mal, wenn Sie eine Änderung vor sich nehmen. Es gibt keinen **Save-Befehl.** Sie können eine Kopie herunterladen, benötigen aber die Microsoft PowerPoint-Desktop-App, um eine lokale Kopie zu bearbeiten.
  
## <a name="shapes"></a>Formen

Word Art and charts can't be inserted in PowerPoint for the web, but PowerPoint for the web displays them as expected. Bearbeiten und formatieren Sie text in der Bearbeitungsansicht. Wenn Sie Texteffekte auf Word Art anwenden möchten, benötigen Sie die Microsoft PowerPoint Desktop-App.
  
## <a name="share"></a>Freigeben

Wenn Ihre Präsentation in einer SharePoint-Dokumentbibliothek gespeichert ist, ist Ihre Präsentation online, und Sie können sie freigeben, indem Sie den betreffenden Personen statt einer E-Mail-Anlage einen Link zu der Präsentation senden. Personen mit den erforderlichen Berechtigungen können die Präsentation in ihrem Webbrowser oder auf ihrem mobilen Gerät anzeigen. Erfahren Sie mehr über das [Freigeben einer Präsentation](https://go.microsoft.com/fwlink/?LinkId=272776).
  
## <a name="slide-show"></a>Slide show

PowerPoint für das Web zeigt Bildschirmpräsentationen in einem Vollbildfenster an. Drücken Sie die LEERTASTE, um die jeweils nächste Folien anzuzeigen. Folienanimationen werden wiedergegeben, als Übergänge werden aber nur "Einblenden" und "Wischen" unterstützt. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="smartart"></a>SmartArt

Sie können SmartArt einfügen, zu einem anderen Layout oder Farbschema wechseln, SmartArt-Formatvorlagen anwenden und Text mit PowerPoint für das Web bearbeiten. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="table-creation-editing-and-formatting"></a>Erstellen, Bearbeiten und Formatieren von Tabellen

Tabellen können in einer PowerPoint für das Web erstellt und bearbeitet werden, und PowerPoint für das Web unterstützt die meisten Tabellenfunktionen. Für erweiterte Funktionen, z. B. das Zusammenführen und Teilen von Zellen, verwenden Sie die PowerPoint Desktop-App. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung einer Präsentation im Browser und in der PowerPoint-Desktop-App](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="tell-me"></a>Sie wünschen...

Wenn Sie etwas in PowerPoint für das Web ausführen müssen, aber nicht wissen, wie, können Sie das Suchfeature Tell Me verwenden, um schnell zu finden, was Sie suchen. Die Funktion versteht, welche Aktion Sie auszuführen versuchen, und hilft Ihnen mithilfe von Vorschlägen, dies schneller zu tun.
  
## <a name="undo-and-redo"></a>"Rückgängig" und "Wiederholen"

Sie können in der jeweils aktuellen Bearbeitungssitzung in der aktiven Präsentation eine unbegrenzte Anzahl von Aktionen rückgängig machen (STRG+Z) und wiederholen (STRG+Y). Wenn bei der Bearbeitungssitzung eine Zeitüberschreitung auftritt oder Sie für länger als 30 Sekunden in die Leseansicht wechseln, wird der Verlauf rückgängig gemachter Vorgänge zurückgesetzt. Erfahren Sie mehr [über Tastenkombinationen in PowerPoint für das Web](https://go.microsoft.com/fwlink/?LinkId=272950).
  
## <a name="view-and-add-comments"></a>Anzeigen und Hinzufügen von Kommentaren

Wenn Ihre Dateien auf SharePoint online oder OneDrive for Business gespeichert werden, können Sie Kommentare hinzufügen, bearbeiten oder löschen. 
  
Wenn Ihre Dateien auf OneDrive gespeichert sind, können Sie Kommentare hinzufügen, bearbeiten oder löschen, während Sie in der Bearbeitungsansicht Kommentare anzeigen und aktualisieren, während Sie in der Leseansicht sind.
  
## <a name="view-and-edit-slide-notes"></a>View and edit slide notes

Mit PowerPoint für das Web können Notizen für jede Folie angezeigt oder ausgeblendet werden. Im Bearbeitungsmodus können Sie auch Notizen hinzufügen.
  
## <a name="wysiwyg-viewing"></a>WYSIWYG-Anzeige

Bearbeiten Sie Ihre Präsentation dahingehend, dass die Darstellung dem gedruckten oder angezeigten Endergebnis ähnelt.
  
## <a name="feature-availability"></a>Verfügbarkeit von Funktionen

Informationen zur Verfügbarkeit von Features in Plänen, eigenständigen Optionen und lokalen Lösungen finden Sie [unter Office der Webdienstbeschreibung](office-online-service-description.md).
