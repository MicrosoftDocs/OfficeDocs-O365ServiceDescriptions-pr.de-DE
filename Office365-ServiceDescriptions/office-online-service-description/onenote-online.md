# <a name="onenote-for-the-web"></a>OneNote für das Web

Verwenden Sie OneNote für das Web (früher OneNote Web App), um Notizen online in einem OneNote-Notizbuch zu machen, das Sie von überall aus hinzufügen und problemlos für andere freigeben können. Alle Kunden können Office-Dateien mit Office für das Web anzeigen und leicht bearbeiten. 
  
## <a name="advanced-collaboration-new-content-shows-as-unread-presence"></a>Advanced collaboration: New content shows as unread, presence

In OneNote für das Web nicht verfügbar. Mit der OneNote-Desktop-App können Sie andere Autoren mithilfe der integrierten Profile in OneNote einfacher identifizieren und authentifizieren. Sie können Notizbuchänderungen nach dem Namen von Autoren suchen und alle aktuellen Bearbeitungen anzeigen, wenn Sie zu einem freigegebenen Notizbuch zurückkehren.
  
## <a name="apply-tags"></a>Apply tags

Mit OneNote für das Web können Sie eine Vielzahl von Tags auf Notizen anwenden, um eine einfache Organisation und Nach- und Nachveranwortung zu erhalten. So können Sie beispielsweise Notizen als Fragen, Aufgaben oder Kontaktinformationen kennzeichnen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="office-add-ins"></a>Office-Add-Ins

OneNote für das Web unterstützt nur Inhalts-Add-Ins für Office. Inhalts-Add-Ins integrieren webbasierte Features als Inhalt, der eingebettet in ein Dokument angezeigt werden kann. Weitere Informationen zu den verschiedenen Typen von Add-Ins für Office finden Sie in dieser [Übersicht](/previous-versions/office/office-2013-resource-kit/jj219429(v=office.15)).
  
## <a name="audio-notes-recording"></a>Audio notes recording

In OneNote für das Web nicht verfügbar. Mit der OneNote-Desktop-App können Sie direkt mit während der Aufzeichnung verfassten Textnotizen verknüpfte Audio- und Videonotizen aufzeichnen. OneNote für das Web gibt keine Audio- und Videoinhalte wieder, aber die Medien bleiben im Notizbuch erhalten, und Sie können Audio- und Videodateien auf Ihren Computer herunterladen, um sie wieder zu spielen. 
  
## <a name="bullets-and-numbering"></a>Aufzählung und Nummerierung

Mit OneNote für das Web können Sie eine Auswahl von drei Aufzählungsformatvorlagen oder fünf Nummerierungsformatvorlagen anwenden. Verwenden Sie die Schaltflächen Einzug erhöhen und Einzug verringern, um die Listenebene für vorhandene aufzählungs- und nummerierte Listen in einem Notizbuch sowie die in OneNote für das Web erstellten zu ändern. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="clipboard"></a>Zwischenablage

Mit OneNote für das Web können Sie Inhalte in einem Notizbuch ausschneiden, kopieren und einfügen. Sie können Text zwischen OneNote für das Web und der OneNote-Desktop-App kopieren und einfügen. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="clipping-experiences"></a>Clipping experiences

In OneNote für das Web nicht verfügbar. Mit der OneNote-Desktop-App können Sie alles auf dem Bildschirm zuschneiden, eine Webseite oder ein gesamtes Dokument an einen Notizbuchabschnitt senden oder Schnellnotizen nach unten heften, die automatisch als Teil Ihres Notizbuchs gespeichert und abgelegt werden.
  
## <a name="co-authoring"></a>Gemeinsame Dokumenterstellung

Mit OneNote für das Web können Sie Notizbücher gleichzeitig mit Personen an anderen Orten bearbeiten, die OneNote für das Web oder die OneNote-Desktop-App verwenden. Erfahren Sie mehr über die [Zusammenarbeit an einem OneNote-Notizbuch](https://go.microsoft.com/fwlink/p/?LinkId=272945).
  
## <a name="create-and-manage-pages-and-sections"></a>Create and manage pages and sections

Mit OneNote für das Web können Sie neue oder vorhandene Notizbücher ganz einfach hinzufügen, indem Sie neue Seiten in Abschnitten oder neue Abschnitte in Notizbüchern erstellen. Seiten und Abschnitte können angepasst, formatiert und in andere Notizbücher verschoben oder kopiert werden. Erfahren Sie mehr [über die Verwendung eines OneNote für das Webnotizbuch](https://go.microsoft.com/fwlink/p/?LinkId=272945).
  
## <a name="download-copy-of-embedded-office-files"></a>Download copy of embedded Office files

Mit OneNote für das Web können Sie Office-Dateien als Anlagen oder Ausdrucke einfügen, um sie als Teil des Notizbuchs zu speichern. Die OneNote 2013-Desktop-App ermöglicht das Einbetten von Excel-Tabellen und Visio-Zeichnungen in Ihre Notizen. Erfahren Sie mehr über das [Einbetten von Microsoft Office-Dateien in OneNote](https://go.microsoft.com/fwlink/p/?LinkId=271843).
  
## <a name="dropbox"></a>Dropbox

Bei Dropbox handelt es sich um einen Dateihostingdienst, der einen Cloudspeicher, eine Dateisynchronisierung, eine persönliche Cloud und Clientsoftware bietet.
  
## <a name="edit-embedded-files"></a>Bearbeiten eingebetteter Dateien

Mit OneNote für das Web können Sie eingebettete Eingereichte auf Ihren Computer herunterladen, um sie zu öffnen. Mit der OneNote-Desktop-App können Sie so gut wie jede Computerdatei an einen beliebigen Teil Ihrer Notizen anfügen, in dem eine Kopie der Datei in Ihrem Notizbuch gespeichert ist. Sie können sogar Outlook-Besprechungsdetails und Outlook-Aufgaben in OneNote einfügen oder eine OneNote-Seite an Outlook senden. Sie können excel-Tabellen und Visio-Diagramme auch direkt in OneNote erstellen oder importieren und ihre Informationen direkt in Ihren Notizen bearbeiten. Eingefügte Dateien werden auf Ihrer Notizenseite als Symbole angezeigt. Doppelklicken Sie auf ein beliebiges Symbol, um die Datei zu öffnen. 
  
## <a name="equations"></a>Formeln

In OneNote für das Web nicht verfügbar. Mit der OneNote-Desktop-App können Sie während einer Besprechung, einer Konferenz oder eines Kurses mathematische Formeln notieren, woraufhin OneNote sofort das Ergebnis für Sie berechnen kann.
  
## <a name="hyperlinks"></a>Hyperlinks

Mit OneNote für das Web können Sie einen Hyperlink zu einer Webadresse einfügen oder einen Link auf ausgewählten Text anwenden. Sie können auch mit der rechten Maustaste auf eine Seitenregisterkarte klicken, um einen direkten Link zu dieser Seite zu kopieren. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="image-optical-character-recognition-ocr"></a>Image optical character recognition (OCR)

In OneNote für das Web nicht verfügbar. Die optische Zeichenerkennung (Optical Character Recognition, OCR) übersetzt Bilder von Text, z. B. gescannte Dokumente, in tatsächliche Textzeichen, die nur in der OneNote-Desktop-App verfügbar sind. 
  
## <a name="ink-viewing"></a>Anzeigen von Freihandobjekten

In OneDrive zeigt OneNote für das Web Freihandeingaben an, Formeln werden jedoch als Platzhalter angezeigt. In SharePoint werden Freihandobjekte angezeigt, Formeln werden als Platzhalter angezeigt, und Formen werden ausgeblendet. Diese können nicht in OneNote für das Web eingefügt oder bearbeitet werden. In OneDrive können Sie Formen, Freihandobjekte und Formelplatzhalter auswählen und löschen. In SharePoint können Sie Platzhalter für diese Objekte auswählen und löschen.
  
## <a name="inking-ink-to-text-and-math-customizable-pens-drawing-tools"></a>Freihand: Freihand-in-Text und Mathe, anpassbare Stifte, Zeichentools

In OneNote für das Web nicht verfügbar. Mit der OneNote-Desktop-App können Sie mit dem Finger, dem Eingabestift oder mit der Maus zeichnen, löschen und bearbeiten. Wenn Sie das Schreiben dem Eingeben vorziehen, kann OneNote Ihre Handschrift konvertieren. Wenn Sie ein Notizbuch in einer SharePoint-Dokumentbibliothek öffnen, werden die Formeln und Freihandobjekte im Notizbuch als Platzhalter angezeigt. 
  
## <a name="linked-notes"></a>Verknüpfte Notizen

Die Möglichkeit, Notizen mit Dateien zu verknüpfen, ist in OneNote für das Web nicht verfügbar, und Links zu Dateien werden ausgeblendet. Aber sie bleiben in den Notizen erhalten, sodass Sie sie mit der OneNote-Desktop-App öffnen können. 
  
## <a name="navigation"></a>Navigation

Mit OneNote für das Web können Sie die Schaltflächen Zurück und Vorwärts auf der Symbolleiste für den Schnellzugriff verwenden, um schnell zwischen Seiten zu springen.
  
## <a name="notebook-management-cross-sectionnotebook-page-filing-section-re-order-createdelete-section-groups"></a>Notebook management: Cross section/notebook page filing, section re-order, create/delete section groups

Mit OneNote für das Web können Sie ziehen und ablegen, um Seiten und Abschnitte neu zu ordnen.  
Erweiterte Verwaltungsfeatures wie das Erstellen, Löschen und Neu ordnen von Abschnittsgruppen sind nur in der OneNote-Desktop-App verfügbar.
  
## <a name="offline-viewing-and-authoring"></a>Offlineansicht und -erstellung

In OneNote für das Web nicht verfügbar. Office für das Web erfordert eine Internetverbindung und einen Webbrowser. Sie benötigen die auf Ihrem Computer installierte OneNote-Desktop-App zum Anzeigen und Bearbeiten eines Dokuments, während die Verbindung mit Office Web Apps Server oder dem Internet getrennt ist.
  
## <a name="outlook-integration-tasks"></a>Outlook integration (tasks)

OneNote für das Web unterstützt keine Befehle, die mit Microsoft Outlook funktionieren, z. B. E-Mail-Seite, Outlook-Aufgaben oder Besprechungsdetails.
  
## <a name="paragraph-formatting"></a>Absatzformatierung

Mit OneNote für das Web können Sie die Absatzausrichtung ändern, den Einzug vom linken Rand vergrößern oder verringern oder die Textrichtung von links nach rechts ändern. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="pictures"></a>Bilder

Mit OneNote für das Web können Sie ein Bild aus einer Datei oder aus Bing Images einfügen. Sie können auch die Bildgröße ändern und Alternativtext hinzufügen. Für erweiterte Bildfunktionen wie Bildschirmausschnitte, gescannte Bilder oder Onlinebilder müssen Sie die OneNote-Desktop-App verwenden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="print"></a>Drucken

Sie können Notizen in OneNote für das Web drucken.
  
## <a name="proofing-tools"></a>Korrekturhilfen

OneNote für das Web überprüft die Rechtschreibung automatisch, wenn Benutzer eingeben, und wendet eine wellige rote Unterstreichung auf falsch geschriebenen Text an. Gängige AutoKorrektur-Aktionen sind enthalten, z. B. die Korrektur häufig vorkommender Rechtschreibfehler oder die Konvertierung von Zeichen in Symbole. Außerdem können Sie die Sprache für die Korrekturhilfe ändern oder die Rechtschreibprüfung für markierten Text deaktivieren. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Rechteverwaltung: Anwenden und Verwenden von IRM- und Kennwortschutz

OneNote für das Web zeigt Notizbücher an, die durch die Verwaltung von Informationsrechten (Information Rights Management, IRM) geschützt sind. Diese Notizbücher können jedoch nicht im Browser bearbeitet werden, und Sie können keine IRM-geschützten Notizbücher in OneNote für das Web erstellen. OneNote für das Web kann keine Dokumente öffnen, die mit einem Kennwort verschlüsselt sind. Erweitere Funktionen für den Dokumentschutz, z. B. das Erstellen IRM-geschützter Notizbücher und das Anwenden von Kennwortschutz, sind nur in der OneNote-Desktop-App verfügbar.
  
## <a name="search-on-page-within-sections"></a>Suchen (auf Seite, innerhalb von Abschnitten)

Mit OneNote für das Web können Sie die Sofortsuche verwenden, um alles zurückrufen, was Sie jemals in OneNote erstellt oder gespeichert haben. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="search-by-tag-title-authordate-audio-notes"></a>Search by: tag, title, author/date, audio notes

In OneNote für das Web nicht verfügbar. Erweiterte Suchfunktionen, z. B. zum Suchen nach Tag, Titel, Autor/Datum und Audiodatei, sind nur in der OneNote-Desktop-App verfügbar.
  
## <a name="share"></a>Freigeben

Wenn Sie Ihr OneNote-Notizbuch in einer SharePoint-Dokumentbibliothek gespeichert haben, ist Ihr OneNote-Notizbuch online. Dies bedeutet, dass Sie es für andere freigeben können, indem Sie den betreffenden Personen statt einer E-Mail-Anlage einen Link zu dem Notizbuch senden. Durch Auswählen des Links können Benutzer Ihre Notizen in ihrem Webbrowser lesen. Erfahren Sie mehr über die [gemeinsame Onlinenutzung von OneNote-Notizen](https://go.microsoft.com/fwlink/p/?LinkID=272945).
  
## <a name="show-or-hide-authors"></a>Show or hide authors

Mit OneNote für das Web können Sie die Autorentags deaktivieren, die angezeigt werden, wenn jemand ein Notizbuch bearbeitet. Dadurch werden die Initialen entfernt, die neben neuen Notizen auf einer Seite angezeigt werden. Autorentags sind standardmäßig aktiviert.
  
## <a name="styles"></a>Formatvorlagen

Mit OneNote für das Web können Sie problemlos Textformatvorlagen für schnelle Formatierungen anwenden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="tables"></a>Tabellen

Mit OneNote für das Web können Sie eine Tabelle einfügen, Tabellentext bearbeiten und einfach die grundlegende Tabellenstruktur bearbeiten, z. B. Zeilen und Spalten hinzufügen oder löschen. Für erweiterte Tabellenfunktionen wie die Konvertierung von Tabellen in Excel-Tabellen, die Schattierung von Zellen, Kopfzeilen oder die Sortierung von Daten in Tabellenzellen müssen Sie die OneNote-Desktop-App verwenden. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="template-support"></a>Template support

In OneNote für das Web nicht verfügbar. Mit der OneNote-Desktop-App können Sie eine Vorlage als Seitenentwurf verwenden, der auf neue Seiten in Ihrem Notizbuch angewendet werden kann, um ihnen einen ansprechenden Hintergrund oder ein einheitlicheres Aussehen oder Layout zu verleihen.
  
## <a name="undo-and-redo"></a>"Rückgängig" und "Wiederholen"

Mithilfe der Tastatur können Sie die zuletzt im aktiven Notizbuch in OneNote für das Web bearbeiteten aktionen (Alt + F7) rückgängig machen (STRG+ Z) oder wiederholen( Alt + F7). Mit OneNote für das Web erhalten Sie einen separaten Rückgängig-Verlauf für jede Im aktiven Notizbuch bearbeitete Seite. Sie können eine unbegrenzte Anzahl von Aktionen pro Seite in der jeweils aktuellen Bearbeitungssitzung rückgängig machen, und zwar bis ein Bild eingefügt oder eine Bearbeitung durch einen anderen Autor empfangen wird. Das Verschieben und Löschen von Seiten kann nicht rückgängig gemacht werden. Erfahren Sie mehr [über OneNote für die Webtastaturverknüpfungen](https://go.microsoft.com/fwlink/p/?LinkId=272944).
  
## <a name="video-playback"></a>Video playback

Videonotizen werden in Notizen beibehalten, können jedoch nicht in OneNote für das Web aufgezeichnet werden. Sie können Videodateien auf Ihren Computer herunterladen und dort wiedergeben. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="view-previous-page-versions"></a>View previous page versions

Mit OneNote für das Web können Sie frühere Seitenversionen einer Seite anzeigen und wiederherstellen, einschließlich der Personen, die sie geschrieben haben und wann. Änderungen gegenüber früheren Versionen einer Seite werden automatisch hervorgehoben. Erfahren Sie mehr über die [Unterschiede zwischen der Verwendung eines Notizbuchs im Browser und in der OneNote-Desktop-App](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="feature-availability"></a>Verfügbarkeit von Funktionen

Informationen zur Verfügbarkeit von Features in Plänen, eigenständigen Optionen und lokalen Lösungen finden Sie unter [Office für die Webdienstbeschreibung](office-online-service-description.md).
